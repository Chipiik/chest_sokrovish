## 🔌 ПОДКЛЮЧЕНИЕ К ARDUINO MEGA

### 1. Grove LED Bar v1.0 (MY9221)

| Провод LED Bar | Цвет провода | Пин Arduino Mega | Примечание |
|----------------|--------------|------------------|------------|
| GND            | Чёрный       | GND              | Земля      |
| VCC            | Красный      | 5V               | Питание 5В |
| D1 (DIN/DATA)  | Белый        | **Digital 2**    | Данные     |
| D2 (CLK)       | Жёлтый       | **Digital 3**    | Тактирование |

**Важно:** 
- Модуль потребляет ~40mA при всех включенных светодиодах
- Можно питать от 5V Arduino

---

### 2. Сервопривод FS5103B

| Провод серво | Цвет           | Пин Arduino Mega | Примечание |
|--------------|----------------|------------------|------------|
| GND          | Коричневый/Чёрный | GND           | Земля      |
| VCC          | Красный        | **5V**           | Питание    |
| SIGNAL       | Оранжевый/Жёлтый | **Digital 9**  | PWM сигнал |

**⚠️ ВНИМАНИЕ:**
- Сервопривод в пиковом режиме потребляет до 500mA!
- При проблемах (дёргания, перезагрузки) используйте **отдельное питание 5V 1A**
- Подключите конденсатор 470-1000 мкФ между VCC и GND серво

**Схема с отдельным питанием:**
```
Power Bank 5V ──┬── VCC серво (красный)
                │
                └── Vin Arduino (если нужно)
                
GND Power Bank ─┬── GND серво (коричневый)
                │
                └── GND Arduino (общая земля!)
```

### 3. Датчик ЭМГ/ЭКГ Bitronics Lab

| Пин модуля | Обозначение | Пин Arduino Mega | Примечание |
|------------|-------------|------------------|------------|
| GND (-)    | Земля       | GND              | Чёрный провод |
| VCC (+)    | Питание     | 5V               | Красный провод |
| OUT        | Сигнал      | **Analog A1**    | Белый провод |

**Настройка усиления (Gain):**
- На модуле есть потенциометр для настройки усиления
- Начальное положение: среднее
- Для лба: усилить больше (по часовой стрелке)
- Для бицепса: меньше (против часовой)

### 4. Датчик ЭЭГ Bitronics Lab

| Пин модуля | Обозначение | Пин Arduino Mega | Примечание |
|------------|-------------|------------------|------------|
| GND (-)    | Земля       | GND              | Чёрный провод |
| VCC (+)    | Питание     | 5V               | Красный провод |
| OUT        | Сигнал      | **Analog A0**    | Белый провод |

**Настройка:**
- Потенциометр Gain выкрутить ближе к максимуму
- ЭЭГ сигнал очень слабый (микровольты)

## 👤 РАЗМЕЩЕНИЕ ЭЛЕКТРОДОВ НА ТЕЛЕ

### Для ЭМГ (мышцы лба):

**Вариант 1 - Лоб (рекомендуется):**
```
        ┌───────────────┐
        │   Электрод 1  │  ← Над левой бровью (2 см выше)
        │       •       │
        │               │
        │   Электрод 2  │  ← Над правой бровью (2 см выше)
        │       •       │
        │               │
        │   Земля       │  ← На виске или за ухом
        │       •       │
        └───────────────┘
```

**Параметры:**
- Расстояние между электродами: 2-3 см
- Обезжирьте кожу спиртом перед наклеиванием
- Используйте гель для ЭКГ для лучшего контакта

**Вариант 2 - Бицепс (альтернатива):**
- Электрод 1: на брюшке бицепса
- Электрод 2: рядом (2-3 см)
- Земля: на локтевом суставе

---

### Для ЭЭГ (бета-ритм):

**Система координат 10-20:**
```
        Нос
         ↑
    Fpz ─┼─ Fz ─ Cz ─ Pz ─ Oz
    (•)  │  (•)  (•)  
         │   
    ─────┴───── Уши
         │
      Референт (за ухом)
```

**Вариант 1 - Fpz (монополярно, рекомендуется):**
- Активный электрод: над переносицей (Fpz)
  - Отступите ~10% от переносицы вверх по лбу
- Референтный: за правым или левым ухом (сосцевидный отросток)
- Земля: на лбу справа или слева

**Вариант 2 - Fz (монополярно):**
- Активный: между Fpz и Cz (центр лба)
- Референт: за ухом
- Земля: на лбу

**Вариант 3 - Биполярно (Fpz-Fz):**
- Электрод 1: Fpz (над переносицей)
- Электрод 2: Fz (центр лба)
- Земля: за ухом

**Важно:**
- Зачешите волосы в стороны от места крепления
- Используйте специальный гель для ЭЭГ
- Зафиксируйте электроды шапочкой или повязкой

## 🚀 ЗАПУСК ТЕСТОВОЙ ПРОГРАММЫ

### Шаг 1: Подготовка оборудования

**Минимальная конфигурация (без датчиков):**
```
Arduino Mega + Grove LED Bar + Сервопривод
```

1. Подключите LED Bar к пинам 2, 3
2. Подключите серво к пину 9
3. Подключите Arduino к компьютеру через USB

### Шаг 2: Загрузка кода

1. Откройте файл `treasure_chest_test.ino` в Arduino IDE
2. Выберите плату: **Инструменты → Плата → Arduino Mega or Mega 2560**
3. Выберите порт: **Инструменты → Порт → COM4** (ваш может отличаться)
4. Нажмите **Проверить** (✓)
5. Нажмите **Загрузить** (→)

### Шаг 3: Открытие Serial Monitor

1. Нажмите **Монитор порта** (лупа в правом верхнем углу)
2. Установите скорость: **115200 бод**
3. Вы увидите приветственное сообщение

### Шаг 4: Использование команд

**Введите `H` и нажмите Enter для получения справки:**

```
=== КОМАНДЫ УПРАВЛЕНИЯ ===
E <0-1023>       - Установить уровень ЭМГ
B <0.0-1.0>      - Установить мощность бета-ритма
E <val> B <val>  - Установить оба значения
R                - Начать калибровку заново
S                - Показать текущее состояние
A                - Автоматический режим (синусоидальная эмуляция)
M                - Ручной режим (вы вводите значения)
H или ?          - Эта справка
=========================
```

### Примеры тестирования:

**Тест 1: Автоматический режим**
```
A              ← Включить авто-эмуляцию
               ← Наблюдайте за изменением значений
S              ← Проверить статус через 30 сек
```

**Тест 2: Ручное управление**
```
M              ← Переключить в ручной режим
E 200          ← Низкий ЭМГ (расслабление)
               ← Крышка должна открываться
E 500          ← Высокий ЭМГ (напряжение)
               ← Крышка должна закрываться
B 0.8          ← Высокий бета-ритм
S              ← Проверить статистику
```

**Тест 3: Быстрая проверка**
```
E 250 B 0.7    ← Установить оба значения
R              ← Перезапустить калибровку
               ← Подождать 30 секунд
               ← Наблюдать за тренингом
```

## 🔋 РАБОТА БЕЗ КОМПЬЮТЕРА

### Ответ: **ДА, устройство может работать автономно!**

### Что нужно для автономной работы:

1. **Источник питания:**
   - Power Bank 5V 2A (рекомендуется)
   - Или батарея 9V с регулятором
   - Или 4xAA батарейный отсек (6V)

2. **Подключение питания:**
   ```
   Power Bank USB ──→ USB кабель ──→ Arduino Mega (USB порт)
   
   ИЛИ
   
   Power Bank 5V ──→ Vin и GND Arduino
   ```

3. **Автономный режим:**
   - Загрузите основную программу `treasure_chest.ino`
   - Подключите все датчики
   - Подайте питание от Power Bank
   - Нажмите кнопку RESET на Arduino

### ⚠️ Важные замечания:

**Без компьютера НЕЛЬЗЯ:**
- ❌ Настроить индивидуальные пороги (калибровка происходит автоматически)
- ❌ Видеть графики сигналов в реальном времени
- ❌ Изменить параметры (скорости, коэффициенты)

**Без компьютера МОЖНО:**
- ✅ Проводить полную калибровку (30 сек)
- ✅ Выполнять тренинг (3 минуты)
- ✅ Получать результат (светодиоды)
- ✅ Использовать устройство на выставке/презентации

### Индикация без компьютера:

| LED Bar | Значение |
|---------|----------|
| 0-10 светодиодов | Прогресс калибровки / угол крышки |
| Все 10 мигают 2× | Калибровка завершена |
| Все 10 горят | УСПЕХ (камни найдены) |
| Красные мигают 3× | НЕУДАЧА |

## ⚙️ НАСТРОЙКА ПАРАМЕТРОВ

### В файле `treasure_chest.ino`:

```cpp
// Время тренинга
const unsigned long TIME_TRAINING = 180000;  // 3 минуты
// Можно изменить: 60000 = 1 мин, 300000 = 5 мин

// Коэффициенты порогов
const float EMG_THRESHOLD_K = 1.3;   // 1.2 = легче, 1.5 = сложнее
const float BETA_THRESHOLD_K = 0.7;  // 0.6 = легче, 0.8 = сложнее

// Процент успеха для бета-ритма
const float BETA_SUCCESS_RATIO = 0.6;  // 60% времени

// Диапазон бета-ритма
const float BETA_FREQ_LOW = 13.0;    // Подростки: 13-18 Гц
const float BETA_FREQ_HIGH = 18.0;
// const float BETA_FREQ_LOW = 12.0;  // Дети: 12-15 Гц
// const float BETA_FREQ_HIGH = 15.0;

// Скорости серво
const float LID_SPEED_OPEN = 0.15;   // Медленнее = плавнее
const float LID_SPEED_CLOSE = 0.30;  // Должно быть ×2 от открытия

// Интервал обновления серво
const unsigned long SERVO_INTERVAL = 200;  // мс
// Если дёргается: увеличить до 300
```

### Рекомендации по настройке:

**Если крышка не открывается:**
- Уменьшите `EMG_THRESHOLD_K` до 1.2
- Проверьте контакт электродов
- Увеличьте Gain на модуле ЭМГ

**Если крышка открывается слишком быстро:**
- Уменьшите `LID_SPEED_OPEN` до 0.10
- Увеличьте `SERVO_INTERVAL` до 300

**Если трудно достичь успеха по бета-ритму:**
- Увеличьте `BETA_THRESHOLD_K` до 0.8
- Уменьшите `BETA_SUCCESS_RATIO` до 0.5

## 🐛 ДИАГНОСТИКА ПРОБЛЕМ

### Проблема 1: Серво дёргается или не двигается

**Причины:**
- Недостаточное питание
- Плохой контакт проводов
- Слишком частое обновление

**Решения:**
1. Используйте отдельный источник питания 5V 1A+ для серво
2. Проверьте надёжность подключения проводов
3. Увеличьте `SERVO_INTERVAL` до 300 мс
4. Добавьте конденсатор 470 мкФ параллельно питанию серво

---

### Проблема 2: LED Bar не светится

**Причины:**
- Неправильное подключение DATA/CLK
- Не установлена библиотека
- Неверные пины

**Решения:**
1. Проверьте подключение: D1→пин 2, D2→пин 3
2. Установите библиотеку Grove_LED_Bar от Seeed Studio
3. Убедитесь, что в коде указаны правильные пины

---

### Проблема 3: Шум на сигналах датчиков

**Причины:**
- Плохой контакт электродов
- Отсутствие заземления
- Помехи от сети 50 Гц

**Решения:**
1. Обезжирьте кожу спиртом
2. Используйте гель для ЭКГ/ЭЭГ
3. Убедитесь, что все компоненты имеют общую землю
4. Увеличьте Gain только если сигнал слишком слабый

---

### Проблема 4: Калибровка проходит, но пороги неверные

**Причины:**
- Испытуемый двигался во время калибровки
- Неправильное размещение электродов

**Решения:**
1. Во время калибровки сидеть абсолютно спокойно
2. Для ЭМГ: лицо полностью расслаблено
3. Для ЭЭГ: глаза открыты, смотреть в одну точку
4. Повторите калибровку (команда `R` в тестовом режиме)

---

### Проблема 5: Устройство работает, но результаты всегда отрицательные

**Причины:**
- Слишком высокие пороги
- Неправильная техника выполнения

**Решения:**
1. Уменьшите коэффициенты порогов
2. Проведите 2-3 тренировочных сеанса
3. Объясните испытуемому технику:
   - Расслабить лоб (представить что он тяжёлый)
   - Решать простые арифметические задачи для бета-ритма
   - Не напрягать лицо при счёте
