# 📦 ПРОЕКТ "СУНДУК СОКРОВИЩ" - Инструкция по сборке и настройке

## 🎯 Описание задачи

Устройство для тренинга биоуправления для коррекции нарушений внимания (СДВГ).

**Часть 1:** Управление крышкой сундука по ЭМГ мышц лба
- Удерживать ЭМГ ниже порога 3 минуты → крышка открывается (0° → 90°)
- Если ЭМГ выше порога → крышка закрывается в 2 раза быстрее
- Сглаживание сигнала (скользящее среднее 500 мс)

**Часть 2:** Детекция бета-ритма ЭЭГ
- Если бета-ритм > порога ≥60% времени → зажечь светодиоды ("драгоценные камни")
- Периоды с высоким ЭМГ не засчитываются

## 🧠 Размещение электродов

### ЭЭГ (Beta-ритм, 13-18 Гц для подростков)
**Отведение:** Fpz, Fz или Cz (монополярно или биполярно)

**Вариант 1 (Fpz - предпочтительно):**
- Активный электрод: на лбу, над переносицей (10% от насона вверх)
- Референтный: за ухом (сосцевидный отросток)

**Вариант 2 (биполярно):**
- Электрод 1: Fpz (над переносицей)
- Электрод 2: Fz (между Fpz и Cz)
- Земля: за ухом

### ЭМГ (мышцы лба)
**Размещение:**
- Электрод 1: над левой бровью (2 см выше)
- Электрод 2: над правой бровью (2 см выше)
- Земля: на виске или за ухом

**Альтернатива (бицепс):**
- Электроды на брюшке бицепса
- Земля: на локте

## ⚙️ Настройка параметров

В начале кода `treasure_chest.ino` можно изменить:

```cpp
// Время тренинга
const unsigned long TIME_TRAINING = 180000; // 3 минуты (по заданию)

// Коэффициенты порогов
const float EMG_THRESHOLD_K = 1.3;   // 1.2 = легче, 1.5 = сложнее
const float BETA_THRESHOLD_K = 0.7;  // 0.6 = легче, 0.8 = сложнее

// Процент успеха для бета-ритма
const float BETA_SUCCESS_RATIO = 0.6; // 60% времени

// Диапазон бета-ритма
const float BETA_FREQ_LOW = 13.0;   // Для подростков: 13-18 Гц
const float BETA_FREQ_HIGH = 18.0;
// const float BETA_FREQ_LOW = 12.0; // Для детей: 12-15 Гц
// const float BETA_FREQ_HIGH = 15.0;

// Скорости серво
const float LID_SPEED_OPEN = 0.15;   // Медленное открытие
const float LID_SPEED_CLOSE = 0.30;  // Быстрое закрытие (×2)
```

## 🚀 Пошаговая инструкция запуска

### Шаг 1: Подготовка оборудования
1. Подключите все компоненты к Arduino Mega по схеме выше
2. Прикрепите сервопривод к крышке сундука
3. Закрепите электроды на голове испытуемого

### Шаг 2: Загрузка кода
1. Откройте `treasure_chest.ino` в Arduino IDE
2. Выберите плату: **Arduino Mega or Mega 2560**
3. Выберите порт: **COM4** (или ваш)
4. Нажмите **Проверить** (✓)
5. Нажмите **Загрузить** (→)

### Шаг 3: Калибровка (30 секунд)
1. Нажмите **RESET** на Arduino
2. Испытуемый сидит спокойно 30 секунд:
   - **Глаза открыты** (для калибровки бета-ритма)
   - **Лицо расслаблено** (для калибровки ЭМГ)
3. LED Bar заполняется от 0 до 10
4. После калибровки все 10 светодиодов мигнут 2 раза

### Шаг 4: Тренинг (3 минуты)
1. **Расслабьте лоб** → крышка медленно открывается
2. **Напрягите лоб** → крышка быстро закрывается
3. **Решайте когнитивные задачи** (счёт, слова) для повышения бета-ритма
4. LED Bar показывает текущий угол открытия крышки

### Шаг 5: Результат
**УСПЕХ** (оба условия выполнены):
- Крышка открыта ≥54° (60% от 90°)
- Бета-ритм > порога ≥60% времени

→ Все светодиоды горят, зелёные мигают 5 раз

**НЕУДАЧА:**
→ Красные светодиоды мигают 3 раза

3. Настройки:
   - Порт: COM4
   - Скорость: 112500
   - Канал КГР: A1
4. Подключить порт

**Графики:**
- Канал 1 (A0): ЭЭГ сигнал
- Канал 2 (A1): ЭМГ сигнал

## 🔧 Диагностика проблем

### Проблема: Серво дёргается
**Решение:**
- Увеличьте интервал обновления: `SERVO_INTERVAL = 300`
- Проверьте питание (серво требует стабильные 5V)
- Используйте отдельный источник питания для серво

### Проблема: Крышка не открывается
**Решение:**
- Уменьшите порог ЭМГ: `EMG_THRESHOLD_K = 1.2`
- Проверьте значения ЭМГ в Serial Monitor
- Убедитесь, что электроды хорошо контактируют с кожей

### Проблема: Бета-ритм не детектируется
**Решение:**
- Увеличьте коэффициент: `BETA_THRESHOLD_K = 0.8`
- Проверьте размещение электродов ЭЭГ
- Убедитесь, что испытуемый решает когнитивные задачи

### Проблема: Шум на сигналах
**Решение:**
- Убедитесь, что все провода экранированы
- Проверьте заземление
- Увеличьте Gain на модулях Bitronics

## 💡 Рекомендации для тренировки

### Перед основным испытанием:
1. Проведите 2-3 тренировочных сеанса
2. Объясните испытуему технику расслабления лба
3. Подберите индивидуальные когнитивные задачи:
   - Арифметический счёт (100 - 7 = 93, 86, 79...)
   - Обратный счёт по Креппелину
   - Подбор синонимов
   - Решение логических задач

### Во время испытания:
- Не требуйте постоянного высокого бета-ритма (это невозможно физиологически)
- Поощряйте короткие периоды концентрации
- Следите за расслаблением лица

## 📚 Теоретическая справка

### Бета-ритм ЭЭГ
- **Частота:** 13-18 Гц (подростки), 12-15 Гц (дети)
- **Локализация:** лобные доли (Fz, Fpz)
- **Состояние:** активное внимание, когнитивная нагрузка
- **Применение:** коррекция СДВГ

### ЭМГ лобных мышц
- **Норма в покое:** низкая амплитуда
- **При напряжении:** рост амплитуды
- **Связь с вниманием:** концентрация → расслабление лица

### БОС-тренинг
- **Курс:** 20-30 сеансов по 15-20 минут
- **Эффект:** улучшение концентрации внимания
- **Механизм:** оперантное обусловливание ритмов ЭЭГ
