# 🚀 Быстрый старт - Пошаговое руководство

## Шаг 1: Подготовка оборудования

### Минимальная конфигурация для тестирования (без датчиков):
- Arduino Mega 2560 × 1
- Grove LED Bar v1.0 × 1
- Сервопривод FS5103B × 1
- USB кабель × 1
- Компьютер с Arduino IDE

### Полная конфигурация (с датчиками):
+ Датчик ЭЭГ Bitronics Lab × 1
+ Датчик ЭМГ/ЭКГ Bitronics Lab × 1
+ Электроды для ЭЭГ и ЭМГ
+ Гель для электродов

---

## Шаг 2: Установка ПО

### 1. Установите Arduino IDE
- Скачайте с официального сайта: https://www.arduino.cc/en/software
- Установите согласно инструкции

### 2. Установите библиотеки
В Arduino IDE:
```
Скетч → Подключить библиотеку → Управлять библиотеками...
```

Найдите и установите:
- ✅ **Grove_LED_Bar** от Seeed Studio
- ✅ **arduinoFFT** от Enrique Condes (только для основной версии)

Библиотека **Servo** уже встроена в Arduino IDE.

---

## Шаг 3: Подключение компонентов

### Для тестирования (без датчиков):

1. **Подключите Grove LED Bar:**
   ```
   LED Bar → Arduino Mega
   GND     → GND
   VCC     → 5V
   D1      → Pin 2
   D2      → Pin 3
   ```

2. **Подключите сервопривод:**
   ```
   Серво   → Arduino Mega
   GND     → GND
   VCC     → 5V ⚠️
   SIGNAL  → Pin 9
   ```

⚠️ **ВАЖНО:** Если серво дёргается, используйте отдельное питание 5V 1A!

---

## Шаг 4: Запуск тестовой программы

### 1. Откройте код
В Arduino IDE откройте файл:
```
src/test/treasure_chest_test.ino
```

### 2. Выберите плату
```
Инструменты → Плата → Arduino Mega or Mega 2560
```

### 3. Выберите порт
```
Инструменты → Порт → COM4 (или ваш)
```

### 4. Загрузите код
Нажмите кнопку **Загрузить** (→)

### 5. Откройте Serial Monitor
- Нажмите лупу в правом верхнем углу
- Установите скорость: **115200 бод**

### 6. Проверьте работу
Вы увидите приветственное сообщение. Введите:
```
HELP
```

---

## Шаг 5: Тестирование команд

### Команды управления:

| Команда | Описание |
|---------|----------|
| `HELP` | Показать справку по командам |
| `STATUS` | Показать текущее состояние системы |
| `AUTO` | Запустить автоматический полный цикл |
| `SUCCESS` | Симуляция успешного выполнения |
| `FAIL_EMG` | Симуляция провала по мышцам |
| `FAIL_EEG` | Симуляция провала по ЭЭГ |
| `RESET` | Перезапустить систему |

### Пример теста:

1. Введите `AUTO` и нажмите Enter
2. Наблюдайте за процессом:
   - Калибровка (30 сек): LED Bar заполняется от 0 до 10
   - Тренинг (3 мин): Серво двигается, LEDs показывают угол
   - Финиш: Результат показывается светодиодами

3. Проверьте статус командой `STATUS`

## Шаг 6: Работа с реальными датчиками

### 1. Подключите датчики

**ЭМГ датчик:**
```
ЭМГ модуль → Arduino Mega
GND (-)    → GND
VCC (+)    → 5V
OUT        → A1
```

**ЭЭГ датчик:**
```
ЭЭГ модуль → Arduino Mega
GND (-)    → GND
VCC (+)    → 5V
OUT        → A0
```

### 2. Разместите электроды

**Для ЭМГ (лоб):**
- Электрод 1: над левой бровью (2 см выше)
- Электрод 2: над правой бровью (2 см выше)
- Земля: на виске или за ухом

**Для ЭЭГ:**
- Активный: Fpz (над переносицей, ~10% от переносицы вверх)
- Референт: за ухом (сосцевидный отросток)
- Земля: на лбу справа или слева

### 3. Загрузите основную программу
Откройте и загрузите:
```
src/main/treasure_chest.ino
```

### 4. Откройте Serial Monitor
Установите скорость: **112500 бод**

### 5. Нажмите RESET на Arduino

### 6. Пройдите калибровку (30 секунд)
- Сидите абсолютно спокойно
- Лицо полностью расслаблено
- Глаза открыты, смотрите в одну точку

### 7. Выполните тренинг (3 минуты)
- Расслабьте лоб → крышка открывается
- Решайте простые арифметические задачи → бета-ритм растёт
- Не напрягайте лицо при счёте!

---

## Шаг 7: Оценка результата

### Критерии успеха:

✅ **УСПЕХ (оба условия выполнены):**
- Крышка открыта ≥54° (60% от 90°)
- Бета-ритм > порога ≥60% времени

**Индикация:** Все 10 светодиодов горят, зелёные мигают 5 раз

❌ **НЕУДАЧА:**
- Хотя бы одно условие не выполнено

**Индикация:** Красные светодиоды мигают 3 раза

## 🔧 Настройка параметров

Если устройство работает неправильно, измените параметры в коде:

### Если крышка не открывается:
```cpp
const float EMG_THRESHOLD_K = 1.2;  // Было 1.3 (уменьшить)
```

### Если серво дёргается:
```cpp
const unsigned long SERVO_INTERVAL = 300;  // Было 200 (увеличить)
```

### Если трудно достичь успеха:
```cpp
const float BETA_THRESHOLD_K = 0.8;      // Было 0.7 (увеличить)
const float BETA_SUCCESS_RATIO = 0.5;    // Было 0.6 (уменьшить)
```

## 🐛 Частые проблемы

| Проблема | Решение |
|----------|---------|
| Ошибка компиляции | Проверьте, что все библиотеки установлены |
| Серво не двигается | Проверьте подключение, используйте отдельное питание |
| LED Bar не светится | Проверьте подключение D1→2, D2→3 |
| Шум на сигналах | Обезжирьте кожу, используйте гель |
| Всегда неудача | Уменьшите пороги, проведите тренировочные сеансы |
