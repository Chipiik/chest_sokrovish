### ASSISTANT
Для реализации этой задачи нам понадобится библиотека **Grove_LED_Bar**, разработанная специально для модулей на чипе MY9221.

### 1. Подготовка

1.  **Установка библиотеки:**
    *   Откройте Arduino IDE.
    *   Зайдите в `Скетч` -> `Подключить библиотеку` -> `Управлять библиотеками...`
    *   В поиске введите `Grove LED Bar`.
    *   Установите библиотеку от **Seeed Studio** (автор: *Seeed Studio*).

2.  **Схема подключения к Arduino Mega:**
    Модуль Grove LED Bar имеет 4 пина (GND, VCC, D1, D2).
    *   **GND** -> GND на Arduino
    *   **VCC** -> 5V на Arduino
    *   **D1 (DIN)** -> Цифровой пин **2** (можно любой другой)
    *   **D2 (CLK)** -> Цифровой пин **3** (можно любой другой)

### 2. Код программы

Этот код запускает таймер на 30 секунд. Светодиоды будут заполняться слева направо по мере истечения времени. Когда 30 секунд пройдут, все светодиоды начнут мигать.

```cpp
#include <Grove_LED_Bar.h>

// Настраиваем пины подключения
// Пин 2: Данные (DIN/D1)
// Пин 3: Тактирование (CLK/D2)
// 0: Порядок свечения (0 = от зеленого к красному, 1 = наоборот)
Grove_LED_Bar bar(2, 3, 0);

unsigned long startTime;
const unsigned long timerDuration = 30000; // 30 секунд в миллисекундах
bool timerFinished = false;

void setup() {
  // Инициализация LED бара
  bar.begin();
  
  // Устанавливаем яркость (0-10)
  bar.setBrightness(10); 
  
  // Сбрасываем все светодиоды в начале
  bar.setBits(0);
  
  startTime = millis(); // Запоминаем время старта
}

void loop() {
  unsigned long currentTime = millis();
  unsigned long elapsedTime = currentTime - startTime;

  if (!timerFinished) {
    // --- ЛОГИКА ТАЙМЕРА ---
    
    // Если прошло больше 30 секунд
    if (elapsedTime >= timerDuration) {
      timerFinished = true;
      bar.setBits(0b1111111111); // Включаем все 10 светодиодов
    } 
    else {
      // Вычисляем, сколько светодиодов должно гореть (от 0 до 10)
      // map(значение, от_мин, от_макс, до_мин, до_макс)
      int level = map(elapsedTime, 0, timerDuration, 0, 10);
      
      // Ограничиваем уровень, чтобы не выйти за пределы 10
      if (level > 10) level = 10;
      
      // Устанавливаем уровень заполнения
      bar.setLevel(level);
    }
  } 
  else {
    // --- ЛОГИКА ПОСЛЕ ОКОНЧАНИЯ ТАЙМЕРА (МИГАНИЕ) ---
    
    bar.setBits(0b1111111111); // Все включены
    delay(300);
    bar.setBits(0);            // Все выключены
    delay(300);
    
    // Если хотите, чтобы таймер перезапускался автоматически, 
    // раскомментируйте строку ниже:
    // resetTimer(); 
  }
}

// Функция для сброса таймера (если нужно перезапускать)
void resetTimer() {
  startTime = millis();
  timerFinished = false;
  bar.setBits(0);
}
```

### Как это работает:
1.  **`bar.setLevel(level)`**: Эта функция из библиотеки автоматически зажигает нужное количество светодиодов подряд.
2.  **`millis()`**: Мы используем системное время Arduino, а не функцию `delay()`, чтобы отсчет был точным и не зависел от времени работы процессора.
3.  **Логика**:
    *   Пока время < 30 сек: мы берем прошедшее время, переводим его в диапазон от 0 до 10 и зажигаем соответствующее число диодов.
    *   Когда время >= 30 сек: включаем все диоды и запускаем цикл мигания.

### Как перезапустить таймер?
В текущем коде таймер отрабатывает один раз и начинает мигать. Если вы хотите, чтобы он сбрасывался при нажатии кнопки (например, подключенной к пину 4), добавьте в `loop()` проверку кнопки и вызов функции `resetTimer()`.

```cpp
#include <Grove_LED_Bar.h>

// Настройка пинов: DATA -> 2, CLOCK -> 3, REVERSE -> 0
Grove_LED_Bar bar(2, 3, 0);

const unsigned long TIMER_MS = 30000;  // Длительность таймера (30 сек)
unsigned long startTime;               // Время запуска
bool isFinished = false;               // Флаг завершения

void setup() {
  bar.begin();          // Инициализация модуля
  bar.setBits(0);       // Выключаем все светодиоды
  startTime = millis(); // Запоминаем момент старта
}

void loop() {
  unsigned long elapsed = millis() - startTime;

  if (isFinished) {
    // === ТАЙМЕР ЗАВЕРШЁН: мигание ===
    bar.setBits(0b1111111111); // Все включены
    delay(300);
    bar.setBits(0);            // Все выключены
    delay(300);
  } 
  else if (elapsed >= TIMER_MS) {
    // === ПРОШЛО 30 СЕКУНД ===
    isFinished = true;
    bar.setBits(0b1111111111); // Зажигаем все светодиоды
  } 
  else {
    // === ИДЁТ ОТСЧЁТ: заполняем шкалу ===
    // Пропорционально вычисляем количество горящих светодиодов (0..10)
    int level = elapsed * 10 / TIMER_MS;
    if (level > 10) level = 10; // Защита от выхода за пределы
    bar.setLevel(level);        // Обновляем индикацию
  }
}
```

### Разбор кода по частям

Этот код реализует **неблокирующий таймер**. Это значит, что отсчёт времени идёт точно, независимо от других действий (например, мигания в конце).

#### 1. Подключение и переменные
```cpp
Grove_LED_Bar bar(2, 3, 0);
```
*   **2**: Пин данных (D1) на Arduino.
*   **3**: Пин тактирования (D2) на Arduino.
*   **0**: Направление (0 = заполняется от зеленого к красному).

```cpp
const unsigned long DURATION = 30000;
unsigned long startTime;
bool isFinished = false;
```
*   **DURATION**: Время таймера в миллисекундах (30000 мс = 30 сек).
*   **startTime**: Момент, когда таймер был запущен.
*   **isFinished**: Флаг-переключатель. `false` — идёт отсчёт, `true` — время вышло.

#### 2. Настройка (`setup`)
Запускается один раз при включении питания:
1.  `bar.begin()` — подготавливает библиотеку к работе.
2.  `bar.setBits(0)` — гасит все светодиоды (очищает шкалу).
3.  `startTime = millis()` — запоминает текущее время системы как точку отсчёта.

#### 3. Основной цикл (`loop`)
Выполняется бесконечно. Внутри есть три состояния:

**Состояние 1: Таймер истёк (Мигание)**
```cpp
if (isFinished) {
  bar.setBits(0b1111111111); // Все 10 диодов ВКЛ
  delay(300);
  bar.setBits(0);            // Все 10 диодов ВЫКЛ
  delay(300);
}
```
Если флаг `isFinished` стал `true`, программа входит в этот блок и бесконечно мигает всеми светодиодами.

**Состояние 2: Финишная черта**
```cpp
else if (elapsed >= DURATION) {
  isFinished = true;
  bar.setBits(0b1111111111);
}
```
Проверяется, прошло ли 30 секунд.
*   Если да: ставим флаг `isFinished = true` (в следующий круг сработает Состояние 1) и зажигаем всю шкалу.

**Состояние 3: Идёт отсчёт**
```cpp
else {
  int level = elapsed * 10 / DURATION;
  if (level > 10) level = 10;
  bar.setLevel(level);
}
```
*   **Математика**: `elapsed * 10 / DURATION` вычисляет пропорцию.
    *   Прошло 0% времени → `level` = 0.

```cpp
const unsigned long DURATION = 30000;
unsigned long startTime;
bool isFinished = false;   // Таймер истёк
bool isBlinking = false;   // Идёт процесс мигания
int blinkCount = 0;        // Счётчик морганий

  bar.begin();
  bar.setBits(0);
  startTime = millis();
}

void loop() {
  unsigned long elapsed = millis() - startTime;

  // 1. Сначала проверяем, не мигаем ли мы уже
  if (isBlinking) {
    bar.setBits(0b1111111111); // ВКЛ
    delay(300);
    bar.setBits(0);            // ВЫКЛ
    delay(300);
    
    blinkCount++;
    
    // Если моргнули 3 раза — выходим из loop() и больше ничего не делаем
    if (blinkCount >= 3) {
      return; 
    }
  }
  // 2. Если таймер истёк, но мигание ещё не началось
  else if (isFinished) {
    isBlinking = true; // Запускаем режим мигания
  }
  // 3. Момент, когда время вышло
  else if (elapsed >= DURATION) {
    isFinished = true;
    bar.setBits(0b1111111111); // Зажигаем всё перед миганием

  else {

    if (level > 10) level = 10;
    bar.setLevel(level);
  }
}
```

### Как это работает:

| Этап | Переменные | Что делает код |
|------|------------|----------------|
| **Отсчёт** | `isFinished = false` | Плавно заполняет шкалу (`setLevel`) |
| **Финиш** | `isFinished = true` | Зажигает все светодиоды, готовится к миганию |
| **Мигание** | `isBlinking = true` | Моргает всеми диодами, считает `blinkCount` |
| **Конец** | `blinkCount >= 3` | Выполняет `return` — выход из `loop()`, программа "засыпает" |

Это потребляет минимум ресурсов и не мешает работе таймера.

*   **Grove LED Bar**: DATA → Пин 2, CLK → Пин 3, VCC → 5V, GND → GND
*   **Сервопривод FS5103B**: Оранжевый (Signal) → Пин 9, Красный (VCC) → 5V, Коричневый (GND) → GND

### Код программы

```cpp
#include <Grove_LED_Bar.h>
#include <Servo.h>

Grove_LED_Bar bar(2, 3, 1);
Servo servo;

// === НАСТРОЙКИ ВРЕМЕНИ ===
const unsigned long BLINK_DURATION = 5000;    // 5 сек мигания
const unsigned long COUNTDOWN_1 = 30000;      // 30 сек обратный отсчет
const unsigned long COUNTDOWN_2 = 60000;      // 60 сек с серво
const int SERVO_START = 0;                    // Начальный угол
const int SERVO_END = 90;                     // Конечный угол

// === ФЛАГИ ===
bool isBrain = true;       // Глобальный флаг разрешения работы
bool isDone = false;       // Флаг полного завершения цикла

// === СОСТОЯНИЯ ===
enum State { BLINK, COUNT1, WAIT, COUNT2, FINISH };
State currentState = BLINK;

// === ПЕРЕМЕННЫЕ ===
unsigned long phaseStart;
int blinkCount = 0;
int servoPos = SERVO_START;

void setup() {
  bar.begin();
  bar.setLevel(0);
  servo.attach(9);
  servo.write(servoPos);
  phaseStart = millis();
}

void loop() {
  // === ГЛОБАЛЬНАЯ ПРОВЕРКА ===
  if (!isBrain) {
    resetAll();
    return;
  }
  
  if (isDone) return;

  unsigned long elapsed = millis() - phaseStart;

  switch (currentState) {
    
    case BLINK:
      if (elapsed >= BLINK_DURATION) {
        currentState = COUNT1;
        phaseStart = millis();
        bar.setLevel(10);
      } else {
        bar.setLevel(10);
        delay(250);
        bar.setLevel(0);
        delay(250);
      }
      break;

    case COUNT1:
      if (elapsed >= COUNTDOWN_1) {
        currentState = WAIT;
        bar.setLevel(0);
      } else {
        int level = 10 - (elapsed * 10 / COUNTDOWN_1);
        if (level < 0) level = 0;
        bar.setLevel(level);
      }
      break;

    case WAIT:
      // Глобальный флаг ожидания (можно использовать для внешних событий)
      // Сейчас просто краткая пауза перед следующим этапом
      if (elapsed >= 1000) {
        currentState = COUNT2;
        phaseStart = millis();
        servoPos = SERVO_START;
        servo.write(servoPos);
      }
      break;

    case COUNT2:
      if (elapsed >= COUNTDOWN_2) {
        currentState = FINISH;
        isDone = true;
      } else {
        // Светодиоды
        int level = 10 - (elapsed * 10 / COUNTDOWN_2);
        if (level < 0) level = 0;
        bar.setLevel(level);
        
        // Серво (плавный поворот на 90 градусов за 60 сек)
        servoPos = map(elapsed, 0, COUNTDOWN_2, SERVO_START, SERVO_END);
        servo.write(servoPos);
      }
      break;

    case FINISH:
      bar.setLevel(0);
      break;
  }
}

// === СБРОС ПРИ isBrain = false ===
void resetAll() {
  bar.setLevel(0);
  servo.write(SERVO_START);
  currentState = BLINK;
  phaseStart = millis();
  blinkCount = 0;
  isDone = false;
```

Сначала определите тип вашего светодиода — это критично:

| Тип | Как определить | Схема подключения |
|-----|---------------|------------------|
| **Common Cathode (общий катод)** | Самый длинный вывод → **минус** | Длинный → GND, R/G/B → пины через резисторы |
| **Common Anode (общий анод)** | Самый длинный вывод → **плюс** | Длинный → 5V, R/G/B → пины через резисторы |

#### Схема для **Common Cathode** (наиболее частый вариант):

```
RGB LED (вид снизу, считаем против часовой от ключа-скоса):
     [1] [2] [3] [4]
      │   │   │   │
      │   │   │   └─ R (Red)   → Пин 5 через 220Ω
      │   │   └───── G (Green) → Пин 6 через 220Ω
      │   └───────── B (Blue)  → Пин 7 через 220Ω
      └───────────── GND       → GND Arduino
```

> ⚠️ **Обязательно** используйте резисторы 220–330Ω на каждом цветовом канале, иначе светодиод сгорит!

#### Если у вас **Common Anode**:
- Длинный вывод → **5V**
- В коде замените `analogWrite(pin, value)` на `analogWrite(pin, 255 - value)`

```cpp
    case 2: r = p*255; g = v*255; b = t_*255; break;
    case 3: r = p*255; g = q*255; b = v*255; break;
    case 4: r = t_*255; g = p*255; b = v*255; break;
```

```cpp
    bar.setLevel(10);
  } else {
    bar.setLevel(10);
    delay(250);
    bar.setLevel(0);
    delay(250);
  }
}

void doCountdown1(unsigned long elapsed) {
  if (elapsed >= COUNTDOWN_1) {
    currentState = WAIT;
    phaseStart = millis();
    bar.setLevel(0);
  } else {
    bar.setLevel(calcLevel(elapsed, COUNTDOWN_1));
  }
}

void doWaitPhase(unsigned long elapsed) {
  if (elapsed >= 1000) {
    currentState = COUNT2;
    phaseStart = millis();
    servo.write(SERVO_START);
  }
}

void doCountdown2(unsigned long elapsed) {
  if (elapsed >= COUNTDOWN_2) {
    currentState = FINISH;
    isDone = true;
    bar.setLevel(0);
    servo.write(SERVO_END);
  } else {
    bar.setLevel(calcLevel(elapsed, COUNTDOWN_2));
    int pos = map(elapsed, 0, COUNTDOWN_2, SERVO_START, SERVO_END);
    servo.write(pos);
  }
}

// === ФИКС: финал — всё выключено ===
void doFinishPhase() {
  bar.setLevel(0);
  // Серво остаётся на 90°, светодиоды выключены
}

// === ВСПОМОГАТЕЛЬНЫЕ ФУНКЦИИ ===

int calcLevel(unsigned long elapsed, unsigned long duration) {
  int level = 10 - (elapsed * 10 / duration);
  return (level < 0) ? 0 : level;
}

void resetAll() {
  bar.setLevel(0);
  servo.write(SERVO_START);
  currentState = BLINK;
  phaseStart = millis();
  isDone = false;
}
```

```cpp
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed < CALIBRATION_TIME) {
    // Сбор данных для калибровки
    float emgVal, eegVal;
    
    // ЭМГ: считываем и усредняем
    if (bitronicsRead(CH_EMG, emgVal)) {
      emgSmoothed = smoothEMG(emgVal);
      static float emgSum = 0;
      static uint16_t emgCnt = 0;
      emgSum += emgSmoothed;
      emgCnt++;
      
      if (millis() - lastEmgTime >= 50) {  // 20 Гц
        lastEmgTime = millis();
        // FFT для ЭЭГ
        if (bitronicsRead(CH_EEG, eegVal)) {
          fftReal[fftIdx] = eegVal;
          fftIdx = (fftIdx + 1) % FFT_SIZE;
          
          if (fftIdx == 0) {  // буфер полон
            float beta = calculateBetaPower(fftReal, FFT_SIZE, betaLow, betaHigh);
            static float betaMax = 0;
            if (beta > betaMax) betaMax = beta;
            betaThreshold = betaMax * BETA_THRESHOLD_MULTIPLIER;
          }
        }
      }
      
      // По окончании калибровки ЭМГ
      if (elapsed >= CALIBRATION_TIME / 2 && emgCnt > 0) {
        emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_MULTIPLIER;
        Serial.print("EMG threshold: ");
        Serial.println(emgThreshold);
      }
    }
    
    // Визуализация прогресса калибровки
    int progress = map(elapsed, 0, CALIBRATION_TIME, 0, 10);
    bar.setLevel(progress);
    
  } else {
    // Калибровка завершена
    isCalibrated = true;
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCount = 0;
    validTimeCount = 0;
    bar.setLevel(10);  // сигнал: готовы
    delay(1000);
    bar.setLevel(0);
    Serial.println(">>> Calibration complete. Training started!");
  }
}

// ==================== ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TRAINING_TIME) {
    currentState = FINISH;
    evaluateResult();
    return;
  }
  
  float emgVal, eegVal;
  
  // === ЧТЕНИЕ ЭМГ И УПРАВЛЕНИЕ КРЫШКОЙ ===
  if (bitronicsRead(CH_EMG, emgVal)) {
    emgSmoothed = smoothEMG(emgVal);
    
    if (emgSmoothed < emgThreshold) {
      // Расслаблен → открываем
      lidAngle += LID_OPEN_SPEED;
      if (lidAngle > LID_TARGET_ANGLE) lidAngle = LID_TARGET_ANGLE;
    } else {
      // Напряжён → закрываем в 2 раза быстрее
      lidAngle -= LID_CLOSE_SPEED;
      if (lidAngle < 0) lidAngle = 0;
    }
    servo.write((int)lidAngle);
  }
  
  // === ЧТЕНИЕ ЭЭГ И АНАЛИЗ БЕТА-РИТМА ===
  if (bitronicsRead(CH_EEG, eegVal)) {
    fftReal[fftIdx] = eegVal;
    fftIdx = (fftIdx + 1) % FFT_SIZE;
    
    if (fftIdx == 0) {  // каждые 64 отсчёта (~0.6 сек)
      float betaPower = calculateBetaPower(fftReal, FFT_SIZE, betaLow, betaHigh);
      validTimeCount++;
      
      // Считаем только если ЭМГ в норме (крышка не опускается)
      if (emgSmoothed < emgThreshold) {
        if (betaPower > betaThreshold) {
          betaAboveCount++;
        }
      }
    }
  }
  
  // === ВИЗУАЛИЗАЦИЯ НА LED BAR ===
  // Показываем прогресс открытия крышки
  int level = map((int)lidAngle, 0, LID_TARGET_ANGLE, 0, 10);
  bar.setLevel(level);
}

// ==================== ФИНИШ ====================
void runFinish() {
  bar.setLevel(0);  // гасим шкалу
  // Серво остаётся на последнем положении
}

void evaluateResult() {
  // Критерий 1: крышка открыта > 60°
  bool lidSuccess = (lidAngle > 60);
  
  // Критерий 2: бета выше порога ≥ 60% валидного времени
  bool betaSuccess = false;
  if (validTimeCount > 0) {
    float percent = (float)betaAboveCount / validTimeCount * 100;
    betaSuccess = (percent >= BETA_SUCCESS_PERCENT);
  }
  
  // Показываем результат
  if (lidSuccess && betaSuccess) {
    // УСПЕХ: зажигаем все светодиоды ("драгоценные камни")
    bar.setLevel(10);
    Serial.println("✓ SUCCESS: Gems revealed!");
  } else {
    // НЕУДАЧА: мигаем красной зоной (последние 7 светодиодов)
    for (int i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);  // только красные
      delay(200);
      bar.setBits(0);
      delay(200);
    }
    Serial.println("✗ Try again: focus better!");
  }
}

// ==================== ВСПОМОГАТЕЛЬНЫЕ ФУНКЦИИ ====================

float smoothEMG(float newValue) {
  // Кольцевой буфер + скользящее среднее
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % 10;
  
  float sum = 0;
  for (uint8_t i = 0; i < 10; i++) sum += emgBuffer[i];
  return sum / 10.0;
}

float calculateBetaPower(float *samples, uint8_t count, float lowHz, float highHz) {
  // Упрощённый расчёт мощности в диапазоне (без полного БПФ)
  // Для точности подключите библиотеку arduinoFFT
  float power = 0;
  
  // Грубая оценка: дисперсия сигнала в окне
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) mean += samples[i];
  mean /= count;
  
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);  // RMS как аппроксимация мощности
}

bool bitronicsRead(uint8_t channel, float &value) {
  // Надёжное чтение с таймаутом
  while (bitronics.available()) bitronics.read();  // очистка буфера
  
  bitronics.write(channel);
  unsigned long start = millis();
  
  while (millis() - start < 100) {
    if (bitronics.available() >= 4) {
      byte *ptr = (byte*)&value;
      for (uint8_t i = 0; i < 4; i++) ptr[i] = bitronics.read();
      return !isnan(value);
    }
  }
  return false;
}

  Serial.print(lidAngle);
  Serial.print(',');
  Serial.print(betaAboveCount);
  Serial.print(',');
  Serial.println(validTimeCount);
}

  bar.setLevel(0);
  servo.write(0);
  currentState = IDLE;
  lidAngle = 0;
  fftIdx = 0;
  emgIdx = 0;
  betaAboveCount = 0;
  validTimeCount = 0;
  isCalibrated = false;

}
```

```cpp
#include <Grove_LED_Bar.h>
#include <Servo.h>

// === АППАРАТНЫЕ НАСТРОЙКИ ===
Grove_LED_Bar bar(2, 3, 1);  // DATA=2, CLK=3, reverse=1
Servo servo;

const int SERVO_PIN = 9;
const int SERVO_OPEN = 90;    // сундук открыт
const int SERVO_CLOSED = 0;   // сундук закрыт

// === ТАЙМИНГИ ===
const unsigned long CALIBRATION_TIME = 30000;   // 30 сек калибровка
const unsigned long TRAINING_TIME = 180000;     // 3 минуты тренинг
const unsigned long EMG_WINDOW_MS = 500;        // окно сглаживания ЭМГ

// === ПАРАМЕТРЫ ===
const float EMG_THRESHOLD_K = 1.5;          // порог ЭМГ = среднее × 1.5
const float BETA_THRESHOLD_K = 0.8;         // порог бета = максимум × 0.8
const float BETA_SUCCESS_RATIO = 0.6;       // бета выше порога ≥60% времени
const float LID_SPEED_OPEN = 0.3;           // скорость открытия (град/100мс)
const float LID_SPEED_CLOSE = 0.6;          // скорость закрытия (в 2× быстрее)

// === КАНАЛЫ BITRONICS ===
const uint8_t CH_EEG = 0;
const uint8_t CH_EMG = 3;

// === ФЛАГИ ===
bool isBrain = true;      // глобальный старт/стоп
bool isCalibrated = false;

// === СОСТОЯНИЯ ===
enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// === ПЕРЕМЕННЫЕ ===
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

// Сглаживание ЭМГ
float emgBuffer[10];      // 10 отсчётов × 50 мс = 500 мс
uint8_t emgIdx = 0;

// Буфер для БПФ (упрощённый)
#define FFT_SIZE 64
float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

// Статистика бета-ритма
uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

// === ПРОТОТИПЫ ===
void bitronicsInit();
bool bitronicsRead(uint8_t ch, float &val);
float smoothEMG(float val);
float calcBetaPower(float *buf, uint8_t n);
void runCalibration();
void runTraining();
void runFinish();
void resetAll();
void setLidAngle(float angle);

// ==================== SETUP ====================
void setup() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(SERVO_PIN);
  setLidAngle(SERVO_CLOSED);
  
  bitronicsInit();
  
  // Инициализация буферов
  for (uint8_t i = 0; i < 10; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
  
  phaseStart = millis();
}

// ==================== LOOP ====================
void loop() {
  if (!isBrain) {
    resetAll();
    return;
  }
  
  if (currentState == FINISH) {
    runFinish();
    return;
  }
  
  switch (currentState) {
    case IDLE:
      bar.setLevel(0);
      setLidAngle(SERVO_CLOSED);
      // Ждём, пока isBrain станет false для сброса, или внешний триггер для старта
      if (isCalibrated) {
        currentState = TRAINING;
        phaseStart = millis();
        betaAboveCnt = 0;
        validTimeCnt = 0;
      }
      break;
      
    case CALIBRATION:
      runCalibration();
      break;
      
    case TRAINING:
      runTraining();
      break;
  }
}

// ==================== КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= CALIBRATION_TIME) {
    isCalibrated = true;
    currentState = IDLE;  // перейдёт в TRAINING в следующем цикле
    bar.setLevel(10);     // сигнал: калибровка завершена
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  float emgVal, eegVal;
  
  // Сбор ЭМГ для порога
  if (bitronicsRead(CH_EMG, emgVal)) {
    float smoothed = smoothEMG(emgVal);
    static float emgSum = 0;
    static uint16_t emgCnt = 0;
    emgSum += smoothed;
    emgCnt++;
    
    // На середине калибровки считаем порог ЭМГ
    if (elapsed >= CALIBRATION_TIME / 2 && emgCnt >= 10) {
      emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
    }
  }
  
  // Сбор ЭЭГ для порога бета-ритма
  if (bitronicsRead(CH_EEG, eegVal)) {
    eegBuffer[eegIdx] = eegVal;
    eegIdx = (eegIdx + 1) % FFT_SIZE;
    
    if (eegIdx == 0) {  // буфер полон
      float beta = calcBetaPower(eegBuffer, FFT_SIZE);
      static float betaMax = 0;
      if (beta > betaMax) betaMax = beta;
      betaThreshold = betaMax * BETA_THRESHOLD_K;
    }
  }
  
  // Визуализация прогресса калибровки на LED bar
  int level = map(elapsed, 0, CALIBRATION_TIME, 0, 10);
  bar.setLevel(level);
}

// ==================== ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TRAINING_TIME) {
    currentState = FINISH;
    return;
  }
  
  float emgVal, eegVal;
  
  // === УПРАВЛЕНИЕ КРЫШКОЙ ПО ЭМГ ===
  if (bitronicsRead(CH_EMG, emgVal)) {
    float emgSmooth = smoothEMG(emgVal);
    
    if (emgSmooth < emgThreshold) {
      // Расслаблен → открываем
      lidAngle += LID_SPEED_OPEN;
      if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
    } else {
      // Напряжён → закрываем в 2× быстрее
      lidAngle -= LID_SPEED_CLOSE;
      if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
    }
    setLidAngle(lidAngle);
  }
  
  // === АНАЛИЗ БЕТА-РИТМА (только если ЭМГ в норме) ===
  if (bitronicsRead(CH_EEG, eegVal)) {
    eegBuffer[eegIdx] = eegVal;
    eegIdx = (eegIdx + 1) % FFT_SIZE;
    
    if (eegIdx == 0) {  // каждые 64 отсчёта
      float beta = calcBetaPower(eegBuffer, FFT_SIZE);
      validTimeCnt++;
      
      // Считаем только если мышцы расслаблены (крышка не опускается)
      if (lidAngle > SERVO_CLOSED + 10) {  // небольшой гистерезис
        if (beta > betaThreshold) {
          betaAboveCnt++;
        }
      }
    }
  }
  
  // === ВИЗУАЛИЗАЦИЯ ПРОГРЕССА НА LED BAR ===
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== ФИНИШ ====================
void runFinish() {
  // Оценка результата
  bool lidSuccess = (lidAngle >= 60);  // крышка открыта >60°
  bool betaSuccess = false;
  
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    // УСПЕХ: зажигаем все светодиоды ("драгоценные камни")
    bar.setLevel(10);
  } else {
    // НЕУДАЧА: мигаем красной зоной (последние 7 светодиодов)
    bar.setBits(0b1111111000);
    delay(300);
    bar.setBits(0);
    delay(300);
    bar.setBits(0b1111111000);
    delay(300);
    bar.setBits(0);
    delay(300);
    bar.setBits(0b1111111000);
  }
  // Серво остаётся на последнем положении
}

// ==================== ВСПОМОГАТЕЛЬНЫЕ ФУНКЦИИ ====================

void bitronicsInit() {
  Serial1.begin(57600);
  delay(100);
  // Инициализация модуля (по примеру вашего кода)
  Serial1.write(0xff);
  delay(200);
  while (Serial1.available()) Serial1.read();  // очистка
}

bool bitronicsRead(uint8_t ch, float &val) {
  while (Serial1.available()) Serial1.read();  // очистка буфера
  Serial1.write(ch);
  
  byte *ptr = (byte*)&val;
  uint8_t readCnt = 0;
  unsigned long start = millis();
  
  while (millis() - start < 50 && readCnt < 4) {
    if (Serial1.available()) {
      ptr[readCnt++] = Serial1.read();
    }
  }
  return (readCnt == 4) && !isnan(val);
}

float smoothEMG(float newVal) {
  emgBuffer[emgIdx] = newVal;
  emgIdx = (emgIdx + 1) % 10;
  
  float sum = 0;
  for (uint8_t i = 0; i < 10; i++) sum += emgBuffer[i];
  return sum / 10.0;
}

float calcBetaPower(float *buf, uint8_t n) {
  // Упрощённая оценка мощности (RMS) в диапазоне бета-ритма
  // Для точности можно подключить arduinoFFT и фильтровать 12-18 Гц
  float mean = 0;
  for (uint8_t i = 0; i < n; i++) mean += buf[i];
  mean /= n;
  
  float power = 0;
  for (uint8_t i = 0; i < n; i++) {
    float d = buf[i] - mean;
    power += d * d;
  }
  return sqrt(power / n);
}

void setLidAngle(float angle) {
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  servo.write((int)angle);
}

  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  currentState = IDLE;
  lidAngle = 0;
  eegIdx = 0;
  emgIdx = 0;
  betaAboveCnt = 0;
  validTimeCnt = 0;
  isCalibrated = false;
  // Сброс порогов при необходимости:
  // emgThreshold = 0;
  // betaThreshold = 0;
}
```

### Как запустить калибровку и тренинг

Поскольку консоль отключена, управление осуществляется через флаг `isBrain`:

```cpp
// В любом месте кода (например, при нажатии кнопки на пине 4):
void checkButton() {
  static bool lastState = HIGH;
  bool curState = digitalRead(4);
  
  if (lastState == HIGH && curState == LOW) {
    isBrain = !isBrain;  // переключить состояние
    if (isBrain && !isCalibrated) {
      currentState = CALIBRATION;
      phaseStart = millis();
    }
    delay(50);  // антидребезг
  }
  lastState = curState;
}
```

Добавьте вызов `checkButton();` в начало `loop()`.

### Логика работы

| Этап | Что происходит |
|------|---------------|
| **IDLE** | Ждёт запуска. При `isBrain=true` и `isCalibrated=true` → переходит в TRAINING |
| **CALIBRATION** | 30 сек замера: вычисляет `emgThreshold` и `betaThreshold`. Прогресс на LED bar |
| **TRAINING** | 3 минуты: ЭМГ управляет крышкой, бета-ритм считается только при расслаблении. Прогресс на LED bar |
| **FINISH** | Оценка: если крышка ≥60° И бета выше порога ≥60% времени → все светодиоды ("камни"). Иначе — 3 моргания красной зоной |

Функции `uniorBegin()` и `uniorRead()` из вашего примера — рабочие. Если моя реализация `bitronicsRead()` не подойдёт под ваш модуль, замените её на вашу, сохранив интерфейс:

```cpp
bool bitronicsRead(uint8_t channel, float &value);
```

## 📦 Полный код программы

```cpp
// ==================== ПОДКЛЮЧЕНИЕ БИБЛИОТЕК ====================
#include <Grove_LED_Bar.h>
#include <Servo.h>

// ==================== НАСТРОЙКИ ПИНОВ ====================
const int PIN_LED_DATA = 2;
const int PIN_LED_CLK  = 3;
const int PIN_SERVO    = 9;
const int PIN_BUTTON   = 4;

// ==================== НАСТРОЙКИ ВРЕМЕНИ ====================
const unsigned long TIME_CALIBRATION = 30000;   // 30 секунд калибровка
const unsigned long TIME_TRAINING    = 60000;   // 1 минута тренинг (изменено!)

// ==================== ПАРАМЕТРЫ БИОУПРАВЛЕНИЯ ====================
const float EMG_THRESHOLD_K     = 1.5;    // порог ЭМГ = среднее × 1.5
const float BETA_THRESHOLD_K    = 0.8;    // порог бета = максимум × 0.8
const float BETA_SUCCESS_RATIO  = 0.6;    // бета выше порога ≥60% времени
const float LID_SPEED_OPEN      = 0.6;    // скорость открытия (град/100мс)
const float LID_SPEED_CLOSE     = 1.2;    // скорость закрытия (в 2× быстрее)

// ==================== ПАРАМЕТРЫ СЕРВО ====================
const int SERVO_CLOSED = 0;     // сундук закрыт
const int SERVO_OPEN   = 90;    // сундук полностью открыт
const int LID_SUCCESS_ANGLE = 60;  // минимальный угол для успеха

// ==================== КАНАЛЫ BITRONICS ====================
const uint8_t CH_EEG = 0;       // канал ЭЭГ
const uint8_t CH_EMG = 3;       // канал ЭМГ

// ==================== БУФЕРЫ ====================
#define FFT_SIZE 64
#define EMG_BUFFER_SIZE 10      // 10 × 50мс = 500мс сглаживание

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);  // reverse=1
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isBrain = true;              // глобальный старт/стоп
bool isCalibrated = false;        // калибровка пройдена

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

// Буферы для обработки сигналов
float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

// Статистика бета-ритма
uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
void initBitronics();
bool readBitronics(uint8_t channel, float &value);
float smoothEMG(float newValue);
float calcBetaPower(float *samples, uint8_t count);
void setLidAngle(float angle);

void checkButton();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();

// ==================== SETUP ====================
void setup() {
  initHardware();
  initBitronics();
  phaseStart = millis();
}

// ==================== LOOP ====================
void loop() {
  // 1. Проверка глобального флага
  if (!isBrain) {
    resetAll();
    return;
  }
  
  // 2. Проверка кнопки
  checkButton();
  
  // 3. Обработка состояния
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
}

// ==================== ИНИЦИАЛИЗАЦИЯ ====================
void initHardware() {
  // LED Bar
  bar.begin();
  bar.setLevel(0);
  
  // Серво
  servo.attach(PIN_SERVO);
  setLidAngle(SERVO_CLOSED);
  
  // Кнопка
  pinMode(PIN_BUTTON, INPUT_PULLUP);
  
  // Инициализация буферов
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
}

void initBitronics() {
  Serial1.begin(57600);
  delay(100);
  
  // Инициализация модуля
  Serial1.write(0xff);
  delay(200);
  
  // Очистка буфера
  while (Serial1.available()) Serial1.read();
}

// ==================== ЧТЕНИЕ С BITRONICS ====================
bool readBitronics(uint8_t channel, float &value) {
  // Очистка входного буфера
  while (Serial1.available()) Serial1.read();
  
  // Запрос канала
  Serial1.write(channel);
  
  // Чтение 4 байт (float)
  byte *ptr = (byte*)&value;
  uint8_t readCnt = 0;
  unsigned long start = millis();
  
  while (millis() - start < 50 && readCnt < 4) {
    if (Serial1.available()) {
      ptr[readCnt++] = Serial1.read();
    }
  }
  
  return (readCnt == 4) && !isnan(value);
}

// ==================== ОБРАБОТКА СИГНАЛОВ ====================
float smoothEMG(float newValue) {
  // Кольцевой буфер + скользящее среднее (500 мс)
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

float calcBetaPower(float *samples, uint8_t count) {
  // Упрощённый расчёт мощности (RMS) в бета-диапазоне
  // Для точности можно добавить arduinoFFT с фильтрацией 12-18 Гц
  
  // 1. Среднее значение
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  // 2. Дисперсия → RMS
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== УПРАВЛЕНИЕ СЕРВО ====================
void setLidAngle(float angle) {
  // Ограничение диапазона
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  
  servo.write((int)angle);
  lidAngle = angle;
}

// ==================== КНОПКА УПРАВЛЕНИЯ ====================
void checkButton() {
  static bool lastState = HIGH;
  bool curState = digitalRead(PIN_BUTTON);
  
  // Обработка нажатия (с антидребезгом)
  if (lastState == HIGH && curState == LOW) {
    delay(50);  // антидребезг
    if (digitalRead(PIN_BUTTON) == LOW) {
      // Переключение isBrain
      isBrain = !isBrain;
      
      // Если включили и калибровка не пройдена — старт калибровки
      if (isBrain && !isCalibrated) {
        currentState = CALIBRATION;
        phaseStart = millis();
        emgThreshold = 0;
        betaThreshold = 0;
      }
    }
  }
  lastState = curState;
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  // Автопереход в тренинг после калибровки
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  // Завершение калибровки
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    currentState = IDLE;  // перейдёт в TRAINING автоматически
    bar.setLevel(10);     // сигнал: готово
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  float emgVal, eegVal;
  
  // --- Сбор данных ЭМГ ---
  if (readBitronics(CH_EMG, emgVal)) {
    float smoothed = smoothEMG(emgVal);
    static float emgSum = 0;
    static uint16_t emgCnt = 0;
    
    emgSum += smoothed;
    emgCnt++;
    
    // Расчёт порога ЭМГ на середине калибровки
    if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 10) {
      emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
    }
  }
  
  // --- Сбор данных ЭЭГ ---
  if (readBitronics(CH_EEG, eegVal)) {
    eegBuffer[eegIdx] = eegVal;
    eegIdx = (eegIdx + 1) % FFT_SIZE;
    
    if (eegIdx == 0) {  // буфер полон (64 отсчёта)
      float beta = calcBetaPower(eegBuffer, FFT_SIZE);
      static float betaMax = 0;
      
      if (beta > betaMax) betaMax = beta;
      betaThreshold = betaMax * BETA_THRESHOLD_K;
    }
  }
  
  // --- Визуализация прогресса ---
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ (1 МИНУТА) ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  // Завершение тренинга
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  float emgVal, eegVal;
  
  // --- Управление крышкой по ЭМГ ---
  if (readBitronics(CH_EMG, emgVal)) {
    float emgSmooth = smoothEMG(emgVal);
    
    if (emgSmooth < emgThreshold) {
      // Расслаблен → открываем
      lidAngle += LID_SPEED_OPEN;
      if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
    } else {
      // Напряжён → закрываем в 2× быстрее
      lidAngle -= LID_SPEED_CLOSE;
      if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
    }
    setLidAngle(lidAngle);
  }
  
  // --- Анализ бета-ритма (только при расслаблении) ---
  if (readBitronics(CH_EEG, eegVal)) {
    eegBuffer[eegIdx] = eegVal;
    eegIdx = (eegIdx + 1) % FFT_SIZE;
    
    if (eegIdx == 0) {  // каждые 64 отсчёта (~0.6 сек)
      float beta = calcBetaPower(eegBuffer, FFT_SIZE);
      validTimeCnt++;
      
      // Считаем только если крышка приподнята (ЭМГ в норме)
      if (lidAngle > SERVO_CLOSED + 10) {
        if (beta > betaThreshold) {
          betaAboveCnt++;
        }
      }
    }
  }
  
  // --- Визуализация прогресса на LED Bar ---
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
  // После оценки ничего не делаем — результат зафиксирован
}

void evaluateResult() {
  // Критерий 1: крышка открыта достаточно
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  // Критерий 2: бета-ритм выше порога ≥60% валидного времени
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  // Показ результата
  if (lidSuccess && betaSuccess) {
    // ✓ УСПЕХ: зажигаем все светодиоды ("драгоценные камни")
    bar.setLevel(10);
  } else {
    // ✗ НЕУДАЧА: мигаем красной зоной 3 раза
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);  // только красные (последние 7)
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
  // Серво остаётся на последнем положении
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  currentState = IDLE;
  lidAngle = 0;
  eegIdx = 0;
  emgIdx = 0;
  betaAboveCnt = 0;
  validTimeCnt = 0;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```

## 🐛 Решение проблем

| Проблема | Возможная причина | Решение |
|----------|------------------|---------|
| Светодиоды не горят | Неправильные пины в `Grove_LED_Bar bar(2,3,1)` | Проверьте подключение D1→2, D2→3 |
| Серво дёргается | Нехватка питания | Подключите серво к внешнему 5В источнику |
| Крышка не двигается | Порог ЭМГ слишком высокий | Уменьшите `EMG_THRESHOLD_K` до 1.2 |
| Нет реакции на кнопку | Не включён INPUT_PULLUP | Проверьте `pinMode(PIN_BUTTON, INPUT_PULLUP)` |
| Bitronics не читается | Перепутаны TX/RX | TX модуля → пин 19 (RX1), RX модуля → пин 18 (TX1) |
| Калибровка не завершается | Нет сигнала с датчиков | Проверьте контакт электродов с кожей, используйте гель |

# 📦 Полный код программы (аналоговые датчики)

```cpp
// ==================== ПОДКЛЮЧЕНИЕ БИБЛИОТЕК ====================
#include <Grove_LED_Bar.h>
#include <Servo.h>

// ==================== НАСТРОЙКИ ПИНОВ ====================
const int PIN_LED_DATA   = 2;
const int PIN_LED_CLK    = 3;
const int PIN_SERVO      = 9;
const int PIN_BUTTON     = 4;
const int PIN_EEG_OUT    = A0;  // аналоговый пин для ЭЭГ
const int PIN_EMG_OUT    = A1;  // аналоговый пин для ЭМГ/ЭКГ

// ==================== НАСТРОЙКИ ВРЕМЕНИ ====================
const unsigned long TIME_CALIBRATION = 30000;   // 30 секунд калибровка
const unsigned long TIME_TRAINING    = 60000;   // 1 минута тренинг

// ==================== ПАРАМЕТРЫ БИОУПРАВЛЕНИЯ ====================
const float EMG_THRESHOLD_K     = 1.5;    // порог ЭМГ = среднее × 1.5
const float BETA_THRESHOLD_K    = 0.8;    // порог бета = максимум × 0.8
const float BETA_SUCCESS_RATIO  = 0.6;    // бета выше порога ≥60% времени
const float LID_SPEED_OPEN      = 0.6;    // скорость открытия (град/100мс)
const float LID_SPEED_CLOSE     = 1.2;    // скорость закрытия (в 2× быстрее)

// ==================== ПАРАМЕТРЫ СЕРВО ====================
const int SERVO_CLOSED = 0;
const int SERVO_OPEN   = 90;
const int LID_SUCCESS_ANGLE = 60;

// ==================== БУФЕРЫ ====================
#define FFT_SIZE 64
#define EMG_BUFFER_SIZE 10

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isBrain = true;
bool isCalibrated = false;

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
int readEEG();
int readEMG();
float smoothEMG(float newValue);
float calcBetaPower(float *samples, uint8_t count);
void setLidAngle(float angle);
void checkButton();
void handleIdle();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();

// ==================== SETUP ====================
void setup() {
  initHardware();
  phaseStart = millis();
}

// ==================== LOOP ====================
void loop() {
  if (!isBrain) {
    resetAll();
    return;
  }
  
  checkButton();
  
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
}

// ==================== ИНИЦИАЛИЗАЦИЯ ====================
void initHardware() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(PIN_SERVO);
  setLidAngle(SERVO_CLOSED);
  
  pinMode(PIN_BUTTON, INPUT_PULLUP);
  
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
}

// ==================== ЧТЕНИЕ АНАЛОГОВЫХ ДАТЧИКОВ ====================
int readEEG() {
  // Читаем 5 раз и усредняем для стабильности
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EEG_OUT);
    delay(2);
  }
  return sum / 5;
}

int readEMG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EMG_OUT);
    delay(2);
  }
  return sum / 5;
}

// ==================== ОБРАБОТКА СИГНАЛОВ ====================
float smoothEMG(float newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

float calcBetaPower(float *samples, uint8_t count) {
  // Расчёт RMS (среднеквадратичное значение) как показатель активности
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== УПРАВЛЕНИЕ СЕРВО ====================
void setLidAngle(float angle) {
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  
  servo.write((int)angle);
  lidAngle = angle;
}

// ==================== КНОПКА УПРАВЛЕНИЯ ====================
void checkButton() {
  static bool lastState = HIGH;
  bool curState = digitalRead(PIN_BUTTON);
  
  if (lastState == HIGH && curState == LOW) {
    delay(50);
    if (digitalRead(PIN_BUTTON) == LOW) {
      isBrain = !isBrain;
      
      if (isBrain && !isCalibrated) {
        currentState = CALIBRATION;
        phaseStart = millis();
        emgThreshold = 0;
        betaThreshold = 0;
      }
    }
  }
  lastState = curState;
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    currentState = IDLE;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  // --- Сбор данных ЭМГ ---
  float smoothed = smoothEMG(emgVal);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 10) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  // --- Сбор данных ЭЭГ ---
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  // --- Визуализация прогресса ---
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  // --- Управление крышкой по ЭМГ ---
  float emgSmooth = smoothEMG(emgVal);
  
  if (emgSmooth < emgThreshold) {
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  setLidAngle(lidAngle);
  
  // --- Анализ бета-ритма (только при расслаблении) ---
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    validTimeCnt++;
    
    if (lidAngle > SERVO_CLOSED + 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
  
  // --- Визуализация прогресса ---
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    bar.setLevel(10);  // все светодиоды = успех
  } else {
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);  // красная зона
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  currentState = IDLE;
  lidAngle = 0;
  eegIdx = 0;
  emgIdx = 0;
  betaAboveCnt = 0;
  validTimeCnt = 0;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```

```cpp
#define EMG_BUFFER_SIZE 10

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isBrain = true;
bool isCalibrated = false;

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
int readEEG();
int readEMG();
float smoothEMG(float newValue);
float calcBetaPower(float *samples, uint8_t count);
void setLidAngle(float angle);
void handleIdle();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();

// ==================== SETUP ====================
void setup() {
  initHardware();
  currentState = CALIBRATION;
  phaseStart = millis();
}

// ==================== LOOP ====================
void loop() {
  if (!isBrain) {
    resetAll();
    return;
  }
  
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
}

// ==================== ИНИЦИАЛИЗАЦИЯ ====================
void initHardware() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(PIN_SERVO);
  setLidAngle(SERVO_CLOSED);
  
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
}

// ==================== ЧТЕНИЕ АНАЛОГОВЫХ ДАТЧИКОВ ====================
int readEEG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EEG_OUT);
    delay(2);
  }
  return sum / 5;
}

int readEMG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EMG_OUT);
    delay(2);
  }
  return sum / 5;
}

// ==================== ОБРАБОТКА СИГНАЛОВ ====================
float smoothEMG(float newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

float calcBetaPower(float *samples, uint8_t count) {
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== УПРАВЛЕНИЕ СЕРВО ====================
void setLidAngle(float angle) {
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  
  servo.write((int)angle);
  lidAngle = angle;
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    currentState = IDLE;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  float smoothed = smoothEMG(emgVal);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 10) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  float emgSmooth = smoothEMG(emgVal);
  
  if (emgSmooth < emgThreshold) {
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  setLidAngle(lidAngle);
  
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    validTimeCnt++;
    
    if (lidAngle > SERVO_CLOSED + 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
  
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    bar.setLevel(10);
  } else {
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  currentState = IDLE;
  lidAngle = 0;
  eegIdx = 0;
  emgIdx = 0;
  betaAboveCnt = 0;
  validTimeCnt = 0;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```

## 🐛 Решение проблем

| Проблема | Причина | Решение |
|----------|---------|---------|
| Датчики не реагируют | Плохой контакт электродов | Обезжирьте кожу спиртом, используйте гель |

| Калибровка не завершается | Нет сигнала с A0/A1 | Проверьте провода OUT→A0, A1 |

## 📍 Правильное размещение электродов

### ЭМГ (мышцы лба) — модуль с проводом OUT→A1
```
        ┌─────────────┐
        │   ЛОБ       │
        ├─────────────┤
        │  ●     ●    │  ← Два электрода ЭМГ
        │  │     │    │     (над бровями, симметрично)
        │  └──┬──┘    │
        │     ●       │  ← Земля (чёрный провод)
        │   (висок)   │
        └─────────────┘
```
- **Два активных электрода**: на 2-3 см выше бровей, по центру или симметрично
- **Земля**: на виске или за ухом
- Обезжирьте кожу спиртом перед наклеиванием!

### ЭЭГ (мозг) — модуль с проводом OUT→A0
```
        ┌─────────────┐
        │   ГОЛОВА    │
        │             │
        │     ●       │  ← ЭЭГ электрод: центр лба (Fpz)
        │     │       │     или макушка (Cz)
        │     ●       │  ← Земля: мочка уха или сосцевидный отросток
        └─────────────┘
```
- **Активный электрод**: центр лба (над переносицей) или макушка
- **Земля**: мочка уха, за ухом или на шее

> ⚠️ **Важно**: Все чёрные провода (земля) должны быть подключены к **одной точке** на теле или соединены между собой.

## 📦 Полный код программы

```cpp
// ==================== ПОДКЛЮЧЕНИЕ БИБЛИОТЕК ====================
#include <Grove_LED_Bar.h>
#include <Servo.h>

// ==================== НАСТРОЙКИ ПИНОВ ====================
const int PIN_LED_DATA   = 2;
const int PIN_LED_CLK    = 3;
const int PIN_SERVO      = 9;
const int PIN_EEG_OUT    = A0;
const int PIN_EMG_OUT    = A1;

// ==================== НАСТРОЙКИ ВРЕМЕНИ ====================
const unsigned long TIME_CALIBRATION = 30000;   // 30 секунд калибровка
const unsigned long TIME_TRAINING    = 60000;   // 1 минута тренинг

// ==================== ПАРАМЕТРЫ БИОУПРАВЛЕНИЯ ====================
const float EMG_THRESHOLD_K     = 1.5;    // порог ЭМГ = среднее × 1.5
const float BETA_THRESHOLD_K    = 0.8;    // порог бета = максимум × 0.8
const float BETA_SUCCESS_RATIO  = 0.6;    // бета выше порога ≥60% времени

// === НОВАЯ СКОРОСТЬ СЕРВО (90° за 60 секунд) ===
const float LID_SPEED_OPEN      = 0.15;   // 90° / 60сек / 10 (100мс) = 0.15 град/100мс
const float LID_SPEED_CLOSE     = 0.5;    // закрывается в 3× быстрее

// ==================== ПАРАМЕТРЫ СЕРВО ====================
const int SERVO_CLOSED = 0;
const int SERVO_OPEN   = 90;
const int LID_SUCCESS_ANGLE = 60;

// ==================== БУФЕРЫ ====================
#define FFT_SIZE 64
#define EMG_BUFFER_SIZE 10

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isBrain = true;
bool isCalibrated = false;

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

unsigned long lastSendTime = 0;
const unsigned long SEND_INTERVAL = 50;  // отправка данных каждые 50 мс

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
int readEEG();
int readEMG();
float smoothEMG(float newValue);
float calcBetaPower(float *samples, uint8_t count);
void setLidAngle(float angle);
void handleIdle();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();
void sendToPC(int eeg, int emg, float emgSmooth, float angle);

// ==================== SETUP ====================
void setup() {
  Serial.begin(115200);  // для связи с Bitronics Studio
  initHardware();
  currentState = CALIBRATION;
  phaseStart = millis();
}

// ==================== LOOP ====================
void loop() {
  if (!isBrain) {
    resetAll();
    return;
  }
  
  // Отправка данных на ПК
  if (millis() - lastSendTime >= SEND_INTERVAL) {
    int eegRaw = readEEG();
    int emgRaw = readEMG();
    float emgSmooth = smoothEMG(emgRaw);
    sendToPC(eegRaw, emgRaw, emgSmooth, lidAngle);
    lastSendTime = millis();
  }
  
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
}

// ==================== ИНИЦИАЛИЗАЦИЯ ====================
void initHardware() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(PIN_SERVO);
  setLidAngle(SERVO_CLOSED);
  
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
}

// ==================== ЧТЕНИЕ АНАЛОГОВЫХ ДАТЧИКОВ ====================
int readEEG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EEG_OUT);
    delay(2);
  }
  return sum / 5;
}

int readEMG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EMG_OUT);
    delay(2);
  }
  return sum / 5;
}

// ==================== ОБРАБОТКА СИГНАЛОВ ====================
float smoothEMG(float newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

float calcBetaPower(float *samples, uint8_t count) {
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== УПРАВЛЕНИЕ СЕРВО ====================
void setLidAngle(float angle) {
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  
  servo.write((int)angle);
  lidAngle = angle;
}

// ==================== ОТПРАВКА ДАННЫХ НА ПК ====================
void sendToPC(int eeg, int emg, float emgSmooth, float angle) {
  // Формат: время,EEG,EMG,EMG_сглаженный,Угол_серво,Порог_ЭМГ
  // Bitronics Studio может читать CSV через Serial
  Serial.print(millis());
  Serial.print(',');
  Serial.print(eeg);
  Serial.print(',');
  Serial.print(emg);
  Serial.print(',');
  Serial.print(emgSmooth, 1);
  Serial.print(',');
  Serial.print(angle, 1);
  Serial.print(',');
  Serial.println(emgThreshold, 1);
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    currentState = IDLE;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  float smoothed = smoothEMG(emgVal);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 10) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  float emgSmooth = smoothEMG(emgVal);
  
  // === УПРАВЛЕНИЕ КРЫШКОЙ ===
  if (emgSmooth < emgThreshold) {
    // Расслаблен → открываем МЕДЛЕННО (90° за 60 сек)
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    // Напряжён → закрываем БЫСТРО
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  setLidAngle(lidAngle);
  
  // === АНАЛИЗ БЕТА-РИТМА ===
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    validTimeCnt++;
    
    if (lidAngle > SERVO_CLOSED + 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
  
  // === ВИЗУАЛИЗАЦИЯ ===
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    bar.setLevel(10);
  } else {
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  currentState = IDLE;
  lidAngle = 0;
  eegIdx = 0;
  emgIdx = 0;
  betaAboveCnt = 0;
  validTimeCnt = 0;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```

### Вариант 2: Через Processing (красивые графики)
```java
// Сохраните как sketch.pde в Processing
import processing.serial.*;

Serial port;
float[] eegData = new float[300];
float[] emgData = new float[300];
int idx = 0;

void setup() {
  size(800, 400);
  String portName = Serial.list()[0];  // выберите ваш порт
  port = new Serial(this, portName, 115200);
  background(0);
}

void draw() {
  if (port.available() > 0) {
    String line = port.readStringUntil('\n');
    if (line != null) {
      String[] parts = split(trim(line), ',');
      if (parts.length >= 3) {
        float eeg = float(parts[1]);
        float emg = float(parts[2]);
        
        eegData[idx] = eeg;
        emgData[idx] = emg;
        idx = (idx + 1) % 300;
      }
    }
  }
  
  background(0);
  
  // График ЭЭГ (синий)
  stroke(0, 100, 255);
  for (int i = 0; i < 299; i++) {
    line(i*2, 200 - eegData[i]/2, (i+1)*2, 200 - eegData[i+1]/2);
  }
  
  // График ЭМГ (красный)
  stroke(255, 50, 50);
  for (int i = 0; i < 299; i++) {
    line(i*2, 400 - emgData[i]/2, (i+1)*2, 400 - emgData[i+1]/2);
  }
  
  // Подписи
  fill(0, 100, 255);
  text("EEG (A0)", 10, 20);
  fill(255, 50, 50);
  text("EMG (A1)", 10, 220);
}
```

```python
# requirements: pyserial, matplotlib
import serial
import matplotlib.pyplot as plt
from collections import deque

ser = serial.Serial('COM3', 115200, timeout=1)  # замените COM3
eeg_data = deque(maxlen=300)
emg_data = deque(maxlen=300)

plt.ion()
fig, (ax1, ax2) = plt.subplots(2, 1)

while True:
    if ser.in_waiting:
        line = ser.readline().decode().strip()
        parts = line.split(',')
        if len(parts) >= 3:
            eeg_data.append(float(parts[1]))
            emg_data.append(float(parts[2]))
            
            ax1.clear()
            ax1.plot(eeg_data, 'b')
            ax1.set_title('EEG')
            
            ax2.clear()
            ax2.plot(emg_data, 'r')
            ax2.set_title('EMG')
```

## ⚙️ Новая скорость серво

| Параметр | Значение | Объяснение |
|----------|----------|------------|
| `LID_SPEED_OPEN` | **0.15** град/100мс | 90° ÷ 60сек ÷ 10 = 0.15 |
| `LID_SPEED_CLOSE` | **0.5** град/100мс | В 3× быстрее при напряжении |
| Время полного открытия | **60 секунд** | Только при непрерывном расслаблении |
| Время закрытия | **~18 секунд** | Если напрячь лоб |

### Расчёт:
```
Открытие: 90° ÷ 0.15°/100мс = 600 × 100мс = 60 000 мс = 1 минута ✅
Закрытие: 90° ÷ 0.5°/100мс = 180 × 100мс = 18 000 мс = 18 секунд
```

## 📦 Обновлённый код (с учётом документации)

```cpp
// ==================== ПОДКЛЮЧЕНИЕ БИБЛИОТЕК ====================
#include <Grove_LED_Bar.h>
#include <Servo.h>
#include <TimerOne.h>  // Рекомендована BiTronics Lab

// ==================== НАСТРОЙКИ ПИНОВ ====================
const int PIN_LED_DATA   = 2;
const int PIN_LED_CLK    = 3;
const int PIN_SERVO      = 9;
const int PIN_EEG_OUT    = A0;  // ЭЭГ
const int PIN_EMG_OUT    = A1;  // ЭМГ (A1 т.к. A0 занят ЭЭГ)

// ==================== НАСТРОЙКИ ВРЕМЕНИ ====================
const unsigned long TIME_CALIBRATION = 30000;   // 30 секунд калибровка
const unsigned long TIME_TRAINING    = 60000;   // 1 минута тренинг

// ==================== ПАРАМЕТРЫ БИОУПРАВЛЕНИЯ ====================
const float EMG_THRESHOLD_K     = 1.5;
const float BETA_THRESHOLD_K    = 0.8;
const float BETA_SUCCESS_RATIO  = 0.6;
const float LID_SPEED_OPEN      = 0.15;   // 90° за 60 секунд
const float LID_SPEED_CLOSE     = 0.5;

// ==================== ПАРАМЕТРЫ СЕРВО ====================
const int SERVO_CLOSED = 0;
const int SERVO_OPEN   = 90;
const int LID_SUCCESS_ANGLE = 60;

// ==================== БУФЕРЫ ====================
#define FFT_SIZE 64
#define EMG_BUFFER_SIZE 10

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isBrain = true;
bool isCalibrated = false;

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

unsigned long lastSendTime = 0;
const unsigned long SEND_INTERVAL = 50;

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
int readEEG();
int readEMG();
float smoothEMG(float newValue);
float calcBetaPower(float *samples, uint8_t count);
void setLidAngle(float angle);
void handleIdle();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();
void sendToPC(int eeg, int emg, float emgSmooth, float angle);

// ==================== SETUP ====================
void setup() {
  Serial.begin(112500);  // ⚠️ СКОРОСТЬ ПО ДОКУМЕНТАЦИИ!
  initHardware();
  currentState = CALIBRATION;
  phaseStart = millis();
  
  // Инициализация TimerOne (опционально)
  // Timer1.initialize(3000);  // 3 мс для ЭЭГ
  // Timer1.attachInterrupt(readSensors);
}

// ==================== LOOP ====================
void loop() {
  if (!isBrain) {
    resetAll();
    return;
  }
  
  // Отправка данных на ПК (для Bitronics Lab)
  if (millis() - lastSendTime >= SEND_INTERVAL) {
    int eegRaw = readEEG();
    int emgRaw = readEMG();
    float emgSmooth = smoothEMG(emgRaw);
    sendToPC(eegRaw, emgRaw, emgSmooth, lidAngle);
    lastSendTime = millis();
  }
  
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
}

// ==================== ИНИЦИАЛИЗАЦИЯ ====================
void initHardware() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(PIN_SERVO);
  setLidAngle(SERVO_CLOSED);
  
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
}

// ==================== ЧТЕНИЕ АНАЛОГОВЫХ ДАТЧИКОВ ====================
int readEEG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EEG_OUT);
    delay(2);
  }
  return sum / 5;
}

int readEMG() {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(PIN_EMG_OUT);
    delay(2);
  }
  return sum / 5;
}

// ==================== ОБРАБОТКА СИГНАЛОВ ====================
float smoothEMG(float newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

float calcBetaPower(float *samples, uint8_t count) {
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== УПРАВЛЕНИЕ СЕРВО ====================
void setLidAngle(float angle) {
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  
  servo.write((int)angle);
  lidAngle = angle;
}

// ==================== ОТПРАВКА ДАННЫХ НА ПК ====================
void sendToPC(int eeg, int emg, float emgSmooth, float angle) {
  // Формат для Bitronics Lab / Serial Plotter
  Serial.print(millis());
  Serial.print(',');
  Serial.print(eeg);
  Serial.print(',');
  Serial.print(emg);
  Serial.print(',');
  Serial.print(emgSmooth, 1);
  Serial.print(',');
  Serial.print(angle, 1);
  Serial.print(',');
  Serial.println(emgThreshold, 1);
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    currentState = IDLE;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  float smoothed = smoothEMG(emgVal);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 10) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  int emgVal = readEMG();
  int eegVal = readEEG();
  
  float emgSmooth = smoothEMG(emgVal);
  
  // === УПРАВЛЕНИЕ КРЫШКОЙ ===
  if (emgSmooth < emgThreshold) {
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  setLidAngle(lidAngle);
  
  // === АНАЛИЗ БЕТА-РИТМА ===
  eegBuffer[eegIdx] = eegVal;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    validTimeCnt++;
    
    if (lidAngle > SERVO_CLOSED + 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
  
  // === ВИЗУАЛИЗАЦИЯ ===
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    bar.setLevel(10);
  } else {
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  currentState = IDLE;
  lidAngle = 0;
  eegIdx = 0;
  emgIdx = 0;
  betaAboveCnt = 0;
  validTimeCnt = 0;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```

## 🎯 Правильное размещение электродов (по документации)

### ЭЭГ (модуль EEG)
```
        ┌─────────────┐
        │   ГОЛОВА    │
        │   (вид сбоку)│
        │             │
        │    ● ●      │  ← 2 сигнальных электрода
        │   (затылок) │     на уровне верхней части ушной раковины
        │      ●      │  ← Опорный электрод (чёрная кнопка)
        │   (за ухом) │     за ушной раковиной
        └─────────────┘
```
- **2 сигнальных**: на затылке, над ушами (с зубцами для лучшего контакта)
- **1 опорный (чёрный)**: за ухом
- **Настройте регуляторы**: Gain (усиление) и Noise (подавление шумов)

### ЭМГ (модуль EMG/ECG)
```
        ┌─────────────┐
        │   БИЦЕПС    │
        │  или ЛОБ*   │
        ├─────────────┤
        │  ●     ●    │  ← 2 сигнальных электрода
        │  │     │    │     вдоль мышечных волокон
        │  └──┬──┘    │     расстояние ~2 см
        │     ●       │  ← Опорный электрод (чёрная кнопка)
        │   (локоть)  │     на локте или запястье
        └─────────────┘
```
- **2 сигнальных**: над мышцей вдоль волокон (~2 см между центрами)
- **1 опорный (чёрный)**: на локте или запястье
- **Настройте регулятор**: Gain (усиление)

> *Для вашей задачи (контроль напряжения лба) можно разместить на лбу, но по документации ЭМГ предназначен для мышц тела.

### 1. Скачайте программу
```
www.bitronicslab.com/guide
```

### 2. Настройки подключения
| Параметр | Значение |
|----------|----------|
| **Порт** | Ваш COM-порт Arduino |
| **Скорость** | **112500** (не 115200!) |
| **КГР порт** | A1 |

### 3. Вкладка "Визуализатор"
- Показывает 4 канала одновременно
- Ось X: время, Ось Y: напряжение (0-5В)

### 4. Триггеры (ЭЭГ и ЭМГ)
```
✓ Включите триггер
□ Порог: высота области (амплитуда)
□ Частота: середина диапазона (Гц)
□ Ширина окна: полуширина области
□ Обратный сигнал: сообщение Arduino если > порога
```

## ⚙️ Настройка регуляторов на модулях

### Модуль ЭЭГ (EEG)
| Регулятор | Назначение | Настройка |
|-----------|-----------|-----------|
| **Gain** | Усиление сигнала | Поверните так, чтобы сигнал был 1-3В в покое |
| **Noise** | Подавление шумов | Увеличьте если много помех (50 Гц от сети) |

### Модуль ЭМГ (EMG/ECG)
| Регулятор | Назначение | Настройка |
|-----------|-----------|-----------|
| **Gain** | Усиление сигнала | Поверните так, чтобы при напряжении мышцы сигнал рос |

> ⚠️ **Важно:** Настройка производится **при подключённых электродах** и запущенной Bitronics Lab!

## 🐛 Решение проблем (по документации)

| Проблема | Причина | Решение |
|----------|---------|---------|
| Нет сигнала | Электроды не контактируют | Обезжирьте кожу, смочите электроды |
| Сигнал "перевёрнут" (ЭКГ) | Электроды перепутаны | Поменяйте сигнальные электроды местами |
| Много шумов (50 Гц) | Помехи от сети | Увеличьте Noise на модуле ЭЭГ |
| Сигнал слишком слабый | Gain слишком низкий | Поверните регулятор Gain по часовой |
| Сигнал зашкаливает | Gain слишком высокий | Поверните регулятор Gain против часовой |
| Bitronics Lab не видит порт | Неправильная скорость | Выберите **112500**, не 115200 |

# ✅ Решение проблемы с baud rate (скоростью Serial)

Вы правы — в **Arduino IDE Serial Monitor** в выпадающем списке нет 112500. Но есть **3 рабочих решения**:

---

## 🔧 Решение 1: Вручную вписать 112500 в Serial Monitor (работает!)

Даже если в списке нет 112500, вы можете **вручную вписать** это значение:

```
1. Откройте Serial Monitor (Инструменты → Монитор порта)
2. Кликните в поле где написано "115200"
3. Сотрите значение и вручную впишите: 112500
4. Нажмите Enter
```

✅ **Arduino IDE примет любое значение**, даже если его нет в списке!

---

## 🔧 Решение 2: Изменить код на 115200 (проще)

Если Bitronics Lab Studio не принимает 112500, измените скорость в коде на стандартную:

```cpp
void setup() {
  Serial.begin(115200);  // ← Изменено с 112500 на 115200
  initHardware();
  currentState = CALIBRATION;
  phaseStart = millis();
}
```

**В Bitronics Lab Studio:**
- Попробуйте вручную вписать **115200** в поле скорости
- Если не принимает → используйте **Решение 3**

---

## 🔧 Решение 3: Использовать Serial Plotter вместо Bitronics Lab

Для графиков достаточно **Arduino IDE Serial Plotter**:

```
1. Загрузите код со скоростью 115200
2. Инструменты → Serial Plotter (или Ctrl+Shift+L)
3. Выберите 115200 из списка
4. Вы увидите 4 графика в реальном времени:
   • EEG (синий)
   • EMG (красный)
   • EMG сглаженный (зелёный)
   • Угол серво (оранжевый)
```

# 🔧 Решение ошибки подключения

Ошибка **"нет прав доступа"** означает, что **COM4 уже занят другой программой**.

## ✅ Что нужно сделать:

### 1. **Закройте Arduino IDE (или Serial Monitor/Plotter)**
Если у вас открыт:
- Arduino IDE
- Serial Monitor (Монитор порта)
- Serial Plotter (Графический плоттер)

**Закройте их полностью!** Пока они открыты, Bitronics Lab не сможет подключиться.

---

### 2. **Проверьте, какой порт использует Arduino**
```
1. Откройте Диспетчер устройств
   (Правый клик на Пуск → Диспетчер устройств)

2. Найдите раздел "Порты (COM и LPT)"

3. Там должно быть что-то вроде:
   "Arduino Mega 2560 (COM4)"
   
4. Запомните номер COM-порта
```

---

### 3. **В Bitronics Lab выберите правильный порт**
```
1. Откройте Bitronics Lab
2. Нажмите кнопку "Настройки" (шестерёнка)
3. В поле "Выбор порта" выберите COM4
4. В поле "Скорость передачи" выберите 112500
   (или 115200, если вы изменили в коде)
5. Нажмите "Применить"
6. Нажмите кнопку "Подключить порт"
```

## 🐛 Если всё ещё не работает:

| Проблема | Решение |
|----------|---------|
| **Не закрывается Arduino IDE** | Полностью закройте программу (Файл → Выход) |
| **Другая программа использует порт** | Закройте все программы, которые могут работать с Arduino |
| **Неверный COM-порт** | Проверьте в Диспетчере устройств, может быть COM3 или COM5 |
| **Плохой USB-кабель** | Попробуйте другой кабель (некоторые кабели только для зарядки) |
| **Драйверы не установлены** | Переустановите драйверы Arduino |

# 🔧 Решение проблемы: устройство подключено, но сенсоры не работают

## ✅ Пошаговая диагностика:

### 1. **Проверьте загруженный скетч**

Bitronics Lab ожидает данные в **специальном формате**. Убедитесь, что на Arduino загружен правильный скетч:

```cpp
// Скетч TwoSensors.ino (из документации BiTronics Lab)
#include <TimerOne.h>

volatile int sensorValue0 = 0;
volatile int sensorValue1 = 0;
unsigned long lastTime1 = 0;

void setup() {
  Serial.begin(112500);  // ⚠️ ВАЖНО: именно 112500!
  Timer1.initialize(3000);  // 3 мс
  Timer1.attachInterrupt(readSensors);
}

void loop() {
  // Отправка данных с A0 каждые 3 мс
  Serial.println(sensorValue0);
  
  // Отправка данных с A1 каждые 30 мс
  if (millis() - lastTime1 >= 30) {
    Serial.println(sensorValue1);
    lastTime1 = millis();
  }
  delay(3);
}

void readSensors() {
  sensorValue0 = analogRead(A0);
  sensorValue1 = analogRead(A1);
}
```

**Скачайте готовый скетч:** www.bitronicslab.com/guide

### 3. **Проверьте подключение модулей**

**Важно!** Модули подключаются **при ВЫКЛЮЧЕННОЙ плате Arduino**!

| Модуль | Куда подключать |
|--------|----------------|
| **ЭЭГ (EEG)** | Аналоговый пин **A0** |
| **ЭМГ/ЭКГ (EMG/ECG)** | Аналоговый пин **A0** |
| **КГР (GSR)** | Аналоговый пин **A1** ⚠️ |
| **Пульс (Pulse)** | Аналоговый пин **A0** |

> ⚠️ **Важно:** Если используете **два модуля одновременно** — один должен быть на A0, другой на A1!

---

### 4. **Проверьте электроды**

#### Для ЭМГ (мышцы):
```
✓ 2 электрода на мышце вдоль волокон (расстояние 2 см)
✓ 1 чёрный электрод на локте/запястье
✓ Кнопки одинакового цвета на сигнальных электродах
✓ Чёрная кнопка на опорном электроде
```

#### Для ЭЭГ (мозг):
```
✓ 2 сигнальных электрода с зубцами на затылке (над ушами)
✓ 1 чёрный электрод за ухом
✓ Ремень плотно прилегает к голове
```

#### Для КГР (пальцы):
```
✓ Электроды на указательном и среднем пальцах
✓ Металлическая часть плотно прилегает к подушечкам
```

### 5. **Настройте регуляторы Gain**

На каждом модуле есть регулятор **Gain** (усиление):

```
1. Откройте вкладку "Визуализатор" в Bitronics Lab
2. Смотрите на график
3. Поворачивайте регулятор Gain:
   • Если сигнал ~0 В → увеличьте Gain (по часовой)
   • Если сигнал ~5 В (зашкаливает) → уменьшите Gain (против часовой)
   • Норма: сигнал 1-3 В в покое
```

**Для модуля ЭЭГ** есть ещё регулятор **Noise** (подавление шумов):
- Увеличьте если много помех (линия "дрожит")

# 🎯 Полный проект: Умный сундук с графиками в Bitronics Lab

Вот **готовый код**, который отправляет данные в Bitronics Lab И управляет сундуком:

```cpp
// ==================== ПОДКЛЮЧЕНИЕ БИБЛИОТЕК ====================
#include <Grove_LED_Bar.h>
#include <Servo.h>
#include <TimerOne.h>

// ==================== НАСТРОЙКИ ПИНОВ ====================
const int PIN_LED_DATA   = 2;
const int PIN_LED_CLK    = 3;
const int PIN_SERVO      = 9;
const int PIN_EEG_OUT    = A0;  // ЭЭГ → A0
const int PIN_EMG_OUT    = A1;  // ЭМГ → A1

// ==================== НАСТРОЙКИ ВРЕМЕНИ ====================
const unsigned long TIME_CALIBRATION = 30000;   // 30 сек калибровка
const unsigned long TIME_TRAINING    = 60000;   // 1 мин тренинг

// ==================== ПАРАМЕТРЫ БИОУПРАВЛЕНИЯ ====================
const float EMG_THRESHOLD_K     = 1.5;
const float BETA_THRESHOLD_K    = 0.8;
const float BETA_SUCCESS_RATIO  = 0.6;
const float LID_SPEED_OPEN      = 0.15;   // 90° за 60 сек
const float LID_SPEED_CLOSE     = 0.5;

// ==================== ПАРАМЕТРЫ СЕРВО ====================
const int SERVO_CLOSED = 0;
const int SERVO_OPEN   = 90;
const int LID_SUCCESS_ANGLE = 60;

// ==================== БУФЕРЫ ====================
#define FFT_SIZE 64
#define EMG_BUFFER_SIZE 10

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isBrain = true;
bool isCalibrated = false;

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
volatile int sensorValue0 = 0;  // Для Bitronics Lab (A0)
volatile int sensorValue1 = 0;  // Для Bitronics Lab (A1)
unsigned long lastTime1 = 0;

unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[FFT_SIZE];
uint8_t eegIdx = 0;

uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
float smoothEMG(float newValue);
float calcBetaPower(float *samples, uint8_t count);
void setLidAngle(float angle);
void handleIdle();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();

// ==================== SETUP ====================
void setup() {
  Serial.begin(112500);  // ⚠️ СКОРОСТЬ ДЛЯ BITRONICS LAB!
  
  initHardware();
  
  // Инициализация TimerOne для отправки данных в Bitronics Lab
  Timer1.initialize(3000);  // 3 мс
  Timer1.attachInterrupt(readSensors);
  
  currentState = CALIBRATION;
  phaseStart = millis();
}

// ==================== LOOP ====================
void loop() {
  if (!isBrain) {
    resetAll();
    return;
  }
  
  // Отправка данных с A1 в Bitronics Lab (каждые 30 мс)
  if (millis() - lastTime1 >= 30) {
    Serial.println(sensorValue1);
    lastTime1 = millis();
  }
  
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
}

// ==================== ЧТЕНИЕ СЕНСОРОВ (Timer Interrupt) ====================
void readSensors() {
  sensorValue0 = analogRead(PIN_EEG_OUT);  // A0 - ЭЭГ
  sensorValue1 = analogRead(PIN_EMG_OUT);  // A1 - ЭМГ
  
  // Отправка данных с A0 в Bitronics Lab (каждые 3 мс)
  Serial.println(sensorValue0);
}

// ==================== ИНИЦИАЛИЗАЦИЯ ====================
void initHardware() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(PIN_SERVO);
  setLidAngle(SERVO_CLOSED);
  
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < FFT_SIZE; i++) eegBuffer[i] = 0;
}

// ==================== ОБРАБОТКА СИГНАЛОВ ====================
float smoothEMG(float newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

float calcBetaPower(float *samples, uint8_t count) {
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== УПРАВЛЕНИЕ СЕРВО ====================
void setLidAngle(float angle) {
  if (angle < SERVO_CLOSED) angle = SERVO_CLOSED;
  if (angle > SERVO_OPEN) angle = SERVO_OPEN;
  
  servo.write((int)angle);
  lidAngle = angle;
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  bar.setLevel(0);
  setLidAngle(SERVO_CLOSED);
  
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    currentState = IDLE;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  // Используем данные из sensorValue1 (ЭМГ)
  float smoothed = smoothEMG(sensorValue1);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 10) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  // Используем данные из sensorValue0 (ЭЭГ)
  eegBuffer[eegIdx] = sensorValue0;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  // Визуализация прогресса калибровки
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  // === УПРАВЛЕНИЕ КРЫШКОЙ ПО ЭМГ ===
  float emgSmooth = smoothEMG(sensorValue1);
  
  if (emgSmooth < emgThreshold) {
    // Расслаблен → открываем МЕДЛЕННО
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    // Напряжён → закрываем БЫСТРО
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  setLidAngle(lidAngle);
  
  // === АНАЛИЗ БЕТА-РИТМА ===
  eegBuffer[eegIdx] = sensorValue0;
  eegIdx = (eegIdx + 1) % FFT_SIZE;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, FFT_SIZE);
    validTimeCnt++;
    
    if (lidAngle > SERVO_CLOSED + 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
  
  // === ВИЗУАЛИЗАЦИЯ НА LED BAR ===
  int level = map((int)lidAngle, SERVO_CLOSED, SERVO_OPEN, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
```

# 🔧 Исправление: дёрганье серво и отсутствие реакции

Проблема в том, что **TimerOne прерывает основной цикл**, а `delay()` блокирует выполнение. Вот исправленный код:

```cpp
// ==================== ПОДКЛЮЧЕНИЕ БИБЛИОТЕК ====================
#include <Grove_LED_Bar.h>
#include <Servo.h>

// ==================== НАСТРОЙКИ ПИНОВ ====================
const int PIN_LED_DATA   = 2;
const int PIN_LED_CLK    = 3;
const int PIN_SERVO      = 9;
const int PIN_EEG_OUT    = A0;
const int PIN_EMG_OUT    = A1;

// ==================== НАСТРОЙКИ ВРЕМЕНИ ====================
const unsigned long TIME_CALIBRATION = 30000;
const unsigned long TIME_TRAINING    = 60000;

// ==================== ПАРАМЕТРЫ БИОУПРАВЛЕНИЯ ====================
const float EMG_THRESHOLD_K     = 1.3;    // УМЕНЬШЕНО для лучшей реакции
const float BETA_THRESHOLD_K    = 0.8;
const float BETA_SUCCESS_RATIO  = 0.6;

// === ИСПРАВЛЕННАЯ СКОРОСТЬ СЕРВО ===
const unsigned long SERVO_UPDATE_INTERVAL = 200;  // Обновление серво каждые 200 мс
const float LID_SPEED_OPEN      = 0.3;   // УМЕНЬШЕНО: плавнее открытие
const float LID_SPEED_CLOSE     = 0.6;   // УМЕНЬШЕНО: плавнее закрытие

// ==================== ПАРАМЕТРЫ СЕРВО ====================
const int SERVO_CLOSED = 0;
const int SERVO_OPEN   = 90;
const int LID_SUCCESS_ANGLE = 60;

// ==================== БУФЕРЫ ====================
#define EMG_BUFFER_SIZE 20  // УВЕЛИЧЕНО: лучшее сглаживание (1 сек)

// ==================== ГЛОБАЛЬНЫЕ ОБЪЕКТЫ ====================
Grove_LED_Bar bar(PIN_LED_DATA, PIN_LED_CLK, 1);
Servo servo;

// ==================== ФЛАГИ И СОСТОЯНИЯ ====================
bool isCalibrated = false;

enum State { IDLE, CALIBRATION, TRAINING, FINISH };
State currentState = IDLE;

// ==================== ПЕРЕМЕННЫЕ ====================
unsigned long phaseStart;
float emgThreshold = 0;
float betaThreshold = 0;
float lidAngle = 0;

float emgBuffer[EMG_BUFFER_SIZE];
uint8_t emgIdx = 0;

float eegBuffer[64];
uint8_t eegIdx = 0;

uint16_t betaAboveCnt = 0;
uint16_t validTimeCnt = 0;

// === НОВЫЕ ПЕРЕМЕННЫЕ ДЛЯ ПЛАВНОЙ РАБОТЫ ===
unsigned long lastServoUpdate = 0;
unsigned long lastDataSend = 0;
int rawEMG = 0;
int rawEEG = 0;
bool isRelaxed = false;

// ==================== ПРОТОТИПЫ ФУНКЦИЙ ====================
void initHardware();
int readSensor(int pin);
float smoothEMG(int newValue);
float calcBetaPower(float *samples, uint8_t count);
void updateServo();
void handleIdle();
void runCalibration();
void runTraining();
void runFinish();
void evaluateResult();
void resetAll();
void sendToPC();

// ==================== SETUP ====================
void setup() {
  Serial.begin(112500);
  
  initHardware();
  
  currentState = CALIBRATION;
  phaseStart = millis();
  
  // Сигнал начала
  bar.setLevel(1);
  delay(200);
  bar.setLevel(0);
}

// ==================== LOOP ====================
void loop() {
  unsigned long currentTime = millis();
  
  // === ЧТЕНИЕ ДАТЧИКОВ (каждый цикл) ===
  rawEMG = readSensor(PIN_EMG_OUT);
  rawEEG = readSensor(PIN_EEG_OUT);
  
  // === ОТПРАВКА В BITRONICS LAB (каждые 30 мс) ===
  if (currentTime - lastDataSend >= 30) {
    sendToPC();
    lastDataSend = currentTime;
  }
  
  // === ОБНОВЛЕНИЕ СЕРВО (каждые 200 мс) ===
  if (currentTime - lastServoUpdate >= SERVO_UPDATE_INTERVAL) {
    updateServo();
    lastServoUpdate = currentTime;
  }
  
  // === МАШИНА СОСТОЯНИЙ ===
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
  
  // === НЕБОЛЬШАЯ ПАУЗА ДЛЯ СТАБИЛЬНОСТИ ===
  delay(5);
}

// ==================== ЧТЕНИЕ СЕНСОРА ====================
int readSensor(int pin) {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(pin);
    delayMicroseconds(100);
  }
  return sum / 5;
}

// ==================== СГЛАЖИВАНИЕ ЭМГ ====================
float smoothEMG(int newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

// ==================== РАСЧЁТ БЕТА-РИТМА ====================
float calcBetaPower(float *samples, uint8_t count) {
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== ОБНОВЛЕНИЕ СЕРВО ====================
void updateServo() {
  float emgSmooth = smoothEMG(rawEMG);
  
  // Проверка: расслаблен или напряжён
  if (emgSmooth < emgThreshold) {
    isRelaxed = true;
    // Открываем МЕДЛЕННО
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    isRelaxed = false;
    // Закрываем БЫСТРЕЕ
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  
  // Ограничение диапазона
  if (lidAngle < 0) lidAngle = 0;
  if (lidAngle > 90) lidAngle = 90;
  
  // Установка угла серво
  servo.write((int)lidAngle);
  
  // Визуализация на LED Bar
  int level = map((int)lidAngle, 0, 90, 0, 10);
  bar.setLevel(level);
}

// ==================== ОТПРАВКА ДАННЫХ ====================
void sendToPC() {
  // Формат для Bitronics Lab: A0 и A1
  Serial.println(rawEEG);
  delayMicroseconds(100);
  Serial.println(rawEMG);
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
    lidAngle = 0;
    servo.write(0);
    bar.setLevel(0);
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  // Сглаживание ЭМГ
  float smoothed = smoothEMG(rawEMG);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  // Расчёт порога ЭМГ на середине калибровки
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 20) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  // Сбор данных ЭЭГ
  eegBuffer[eegIdx] = rawEEG;
  eegIdx = (eegIdx + 1) % 64;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, 64);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  // Прогресс калибровки
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  // Анализ бета-ритма
  eegBuffer[eegIdx] = rawEEG;
  eegIdx = (eegIdx + 1) % 64;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, 64);
    validTimeCnt++;
    
    // Считаем только если расслаблен
    if (isRelaxed && lidAngle > 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    bar.setLevel(10);
  } else {
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  servo.write(0);
  lidAngle = 0;
  currentState = IDLE;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```

```cpp
  currentState = CALIBRATION;
  phaseStart = millis();
  bar.setLevel(1);
  delay(200);
  bar.setLevel(0);
}

// ==================== LOOP ====================
void loop() {
  unsigned long currentTime = millis();
  
  rawEMG = readSensor(PIN_EMG_OUT);
  rawEEG = readSensor(PIN_EEG_OUT);
  
  if (currentTime - lastDataSend >= 30) {
    sendToPC();
    lastDataSend = currentTime;
  }
  
  if (currentTime - lastServoUpdate >= SERVO_UPDATE_INTERVAL) {
    updateServo();
    lastServoUpdate = currentTime;
  }
  
  switch (currentState) {
    case IDLE:       handleIdle();       break;
    case CALIBRATION: runCalibration();  break;
    case TRAINING:   runTraining();      break;
    case FINISH:     runFinish();        break;
  }
  
  delay(5);
}

// ==================== ФУНКЦИЯ ИНИЦИАЛИЗАЦИИ (БЫЛА ПРОПУЩЕНА!) ====================
void initHardware() {
  bar.begin();
  bar.setLevel(0);
  
  servo.attach(PIN_SERVO);
  servo.write(SERVO_CLOSED);
  lidAngle = SERVO_CLOSED;
  
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) emgBuffer[i] = 0;
  for (uint8_t i = 0; i < 64; i++) eegBuffer[i] = 0;
}

// ==================== ЧТЕНИЕ СЕНСОРА ====================
int readSensor(int pin) {
  int sum = 0;
  for (uint8_t i = 0; i < 5; i++) {
    sum += analogRead(pin);
    delayMicroseconds(100);
  }
  return sum / 5;
}

// ==================== СГЛАЖИВАНИЕ ЭМГ ====================
float smoothEMG(int newValue) {
  emgBuffer[emgIdx] = newValue;
  emgIdx = (emgIdx + 1) % EMG_BUFFER_SIZE;
  
  float sum = 0;
  for (uint8_t i = 0; i < EMG_BUFFER_SIZE; i++) {
    sum += emgBuffer[i];
  }
  return sum / EMG_BUFFER_SIZE;
}

// ==================== РАСЧЁТ БЕТА-РИТМА ====================
float calcBetaPower(float *samples, uint8_t count) {
  float mean = 0;
  for (uint8_t i = 0; i < count; i++) {
    mean += samples[i];
  }
  mean /= count;
  
  float power = 0;
  for (uint8_t i = 0; i < count; i++) {
    float diff = samples[i] - mean;
    power += diff * diff;
  }
  
  return sqrt(power / count);
}

// ==================== ОБНОВЛЕНИЕ СЕРВО ====================
void updateServo() {
  float emgSmooth = smoothEMG(rawEMG);
  
  if (emgSmooth < emgThreshold) {
    isRelaxed = true;
    lidAngle += LID_SPEED_OPEN;
    if (lidAngle > SERVO_OPEN) lidAngle = SERVO_OPEN;
  } else {
    isRelaxed = false;
    lidAngle -= LID_SPEED_CLOSE;
    if (lidAngle < SERVO_CLOSED) lidAngle = SERVO_CLOSED;
  }
  
  if (lidAngle < 0) lidAngle = 0;
  if (lidAngle > 90) lidAngle = 90;
  
  servo.write((int)lidAngle);
  
  int level = map((int)lidAngle, 0, 90, 0, 10);
  bar.setLevel(level);
}

// ==================== ОТПРАВКА ДАННЫХ ====================
void sendToPC() {
  Serial.println(rawEEG);
  delayMicroseconds(100);
  Serial.println(rawEMG);
}

// ==================== СОСТОЯНИЕ: IDLE ====================
void handleIdle() {
  if (isCalibrated) {
    currentState = TRAINING;
    phaseStart = millis();
    betaAboveCnt = 0;
    validTimeCnt = 0;
    lidAngle = 0;
    servo.write(0);
    bar.setLevel(0);
  }
}

// ==================== СОСТОЯНИЕ: КАЛИБРОВКА ====================
void runCalibration() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_CALIBRATION) {
    isCalibrated = true;
    bar.setLevel(10);
    delay(500);
    bar.setLevel(0);
    return;
  }
  
  float smoothed = smoothEMG(rawEMG);
  static float emgSum = 0;
  static uint16_t emgCnt = 0;
  
  emgSum += smoothed;
  emgCnt++;
  
  if (elapsed >= TIME_CALIBRATION / 2 && emgCnt >= 20) {
    emgThreshold = (emgSum / emgCnt) * EMG_THRESHOLD_K;
  }
  
  eegBuffer[eegIdx] = rawEEG;
  eegIdx = (eegIdx + 1) % 64;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, 64);
    static float betaMax = 0;
    
    if (beta > betaMax) betaMax = beta;
    betaThreshold = betaMax * BETA_THRESHOLD_K;
  }
  
  int level = map(elapsed, 0, TIME_CALIBRATION, 0, 10);
  bar.setLevel(level);
}

// ==================== СОСТОЯНИЕ: ТРЕНИНГ ====================
void runTraining() {
  unsigned long elapsed = millis() - phaseStart;
  
  if (elapsed >= TIME_TRAINING) {
    currentState = FINISH;
    return;
  }
  
  eegBuffer[eegIdx] = rawEEG;
  eegIdx = (eegIdx + 1) % 64;
  
  if (eegIdx == 0) {
    float beta = calcBetaPower(eegBuffer, 64);
    validTimeCnt++;
    
    if (isRelaxed && lidAngle > 10) {
      if (beta > betaThreshold) {
        betaAboveCnt++;
      }
    }
  }
}

// ==================== СОСТОЯНИЕ: ФИНИШ ====================
void runFinish() {
  evaluateResult();
}

void evaluateResult() {
  bool lidSuccess = (lidAngle >= LID_SUCCESS_ANGLE);
  
  bool betaSuccess = false;
  if (validTimeCnt > 0) {
    float ratio = (float)betaAboveCnt / validTimeCnt;
    betaSuccess = (ratio >= BETA_SUCCESS_RATIO);
  }
  
  if (lidSuccess && betaSuccess) {
    bar.setLevel(10);
  } else {
    for (uint8_t i = 0; i < 3; i++) {
      bar.setBits(0b1111111000);
      delay(300);
      bar.setBits(0);
      delay(300);
    }
  }
}

// ==================== СБРОС ====================
void resetAll() {
  bar.setLevel(0);
  servo.write(0);
  lidAngle = 0;
  currentState = IDLE;
  isCalibrated = false;
  emgThreshold = 0;
  betaThreshold = 0;
}
```
