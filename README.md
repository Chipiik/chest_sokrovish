# 📦 ПРОЕКТ "СУНДУК СОКРОВИЩ" - Полная документация

## 🎯 Описание задачи

Устройство для тренинга биоуправления для коррекции нарушений внимания (СДВГ).

**Часть 1:** Управление крышкой сундука по ЭМГ мышц лба
- Удерживать ЭМГ ниже порога 3 минуты → крышка открывается (0° → 90°)
- Если ЭМГ выше порога → крышка закрывается в 2 раза быстрее
- Сглаживание сигнала (скользящее среднее 500 мс)

**Часть 2:** Детекция бета-ритма ЭЭГ
- Если бета-ритм > порога ≥60% времени → зажечь светодиоды ("драгоценные камни")
- Периоды с высоким ЭМГ не засчитываются

## 🔌 Схема подключения

### Arduino Mega → Grove LED Bar v1.0
| LED Bar | Arduino Mega |
|---------|--------------|
| GND     | GND          |
| VCC     | 5V           |
| D1 (DIN)| Pin 2        |
| D2 (CLK)| Pin 3        |

### Arduino Mega → Сервопривод FS5103B
| Сервопровод | Цвет | Arduino Mega |
|-------------|------|--------------|
| GND         | Коричневый/Чёрный | GND |
| VCC         | Красный | 5V ⚠️ |
| SIGNAL      | Оранжевый/Жёлтый | Pin 9 |

⚠️ **ВАЖНО:** Сервопривод может потреблять до 500mA! При проблемах используйте отдельное питание 5V 1A+.

### Arduino Mega → Bitronics Lab EEG
| EEG модуль | Arduino Mega |
|------------|--------------|
| GND (-)    | GND          |
| VCC (+)    | 5V           |
| OUT        | A0           |

### Arduino Mega → Bitronics Lab EMG/ECG
| EMG модуль | Arduino Mega |
|------------|--------------|
| GND (-)    | GND          |
| VCC (+)    | 5V           |
| OUT        | A1           |

## 🚀 Быстрый старт

### Вариант 1: Тестирование БЕЗ датчиков (рекомендуется сначала!)

1. **Подключите:**
   - Grove LED Bar к пинам 2, 3
   - Сервопривод к пину 9
   - Arduino к компьютеру через USB

2. **Загрузите:** `treasure_chest_test.ino`

3. **Откройте Serial Monitor** (115200 бод)

4. **Введите команды:**
   ```
   H              ← Получить справку
   A              ← Автоматический режим
   S              ← Показать статус
   E 200          ← Низкий ЭМГ (крышка открывается)
   E 500          ← Высокий ЭМГ (крышка закрывается)
   R              ← Перезапустить
   ```

### Вариант 2: Работа с реальными датчиками

1. **Подключите все компоненты** по схеме выше

2. **Разместите электроды:**
   - **ЭМГ:** над левой и правой бровью (2 см выше)
   - **ЭЭГ:** Fpz (над переносицей) + за ухом (референт)

3. **Загрузите:** `treasure_chest.ino`

4. **Откройте Bitronics Lab** или Serial Monitor (112500 бод)

5. **Нажмите RESET** на Arduino

6. **Пройдите калибровку** (30 секунд):
   - Сидите спокойно
   - Лицо расслаблено
   - Глаза открыты

7. **Выполните тренинг** (3 минуты):
   - Расслабьте лоб → крышка открывается
   - Решайте арифметические задачи → бета-ритм растёт

## 🔋 Работа без компьютера

**Ответ: ДА, устройство может работать автономно от Power Bank!**

### Что нужно:
- Power Bank 5V 2A (или батарейный отсек 4xAA)
- Подключить к USB порту Arduino или к Vin+GND

### Как работает:
1. Загрузите `treasure_chest.ino`
2. Подключите все датчики
3. Подайте питание от Power Bank
4. Нажмите RESET
5. Устройство автоматически проведёт калибровку и тренинг

### Индикация результата:
- ✅ **УСПЕХ:** Все 10 светодиодов горят, зелёные мигают 5 раз
- ❌ **НЕУДАЧА:** Красные светодиоды мигают 3 раза

## 📊 Критерии успеха

| Условие | Требование | Индикация |
|---------|-----------|-----------|
| **Крышка** | Открыта ≥54° (60% от 90°) | LED Bar показывает угол |
| **Бета-ритм** | > порога ≥60% времени | Зелёные светодиоды |
| **Общий успех** | Оба условия выполнены | Все светодиоды горят |

```cpp
// Время тренинга
const unsigned long TIME_TRAINING = 180000; // 3 минуты

// Коэффициенты порогов (1.2 = легче, 1.5 = сложнее)

// Диапазон бета-ритма

// const float BETA_FREQ_LOW = 12.0;  // Дети: 12-15 Гц

// Скорости серво
const float LID_SPEED_OPEN = 0.15;   // Медленное открытие
```

## 🐛 Диагностика проблем

| Проблема | Решение |
|----------|---------|
| Серво дёргается | Отдельное питание 5V 1A+, увеличить SERVO_INTERVAL до 300 |
| Крышка не открывается | Уменьшить EMG_THRESHOLD_K до 1.2, проверить контакт электродов |
| LED Bar не светится | Проверить подключение D1→2, D2→3, установить библиотеку |
| Шум на сигналах | Обезжирить кожу, использовать гель, проверить заземление |
| Всегда неудача | Провести тренировочные сеансы, уменьшить пороги |
