# 📦 ПРОЕКТ "СУНДУК СОКРОВИЩ" - Биоуправление для коррекции внимания (СДВГ)

## 🎯 Описание проекта

Устройство для тренинга биоуправления, предназначенное для коррекции нарушений внимания (СДВГ) у детей и подростков.

### Задачи устройства:

**Часть 1: Управление крышкой по ЭМГ (мышцы лба)**
- Удерживать ЭМГ ниже порога 3 минуты → крышка открывается (0° → 90°)
- Если ЭМГ выше порога → крышка закрывается в 2 раза быстрее
- Сглаживание сигнала скользящим средним (500 мс)

**Часть 2: Детекция бета-ритма ЭЭГ (13-18 Гц)**
- Если бета-ритм > порога ≥60% времени → зажечь светодиоды ("драгоценные камни")
- Периоды с высоким ЭМГ не засчитываются

## 🔌 Оборудование

### Необходимые компоненты:

| Компонент | Модель | Количество |
|-----------|--------|------------|
| Плата Arduino | Arduino Mega 2560 | 1 |
| LED индикатор | Grove LED Bar v1.0 (MY9221) | 1 |
| Сервопривод | FS5103B | 1 |
| Датчик ЭЭГ | Bitronics Lab EEG | 1 |
| Датчик ЭМГ | Bitronics Lab EMG/ECG | 1 |
| Кабели | Grove/Jumper wires | несколько |

### Требуемые библиотеки Arduino:

1. **Grove_LED_Bar** (Seeed Studio) - установка через Library Manager
2. **Servo** (встроена в Arduino IDE)
3. **arduinoFFT** (Enrique Condes) - только для основной версии

## 🚀 Быстрый старт

### Вариант 1: Тестирование БЕЗ датчиков (рекомендуется!)

1. **Подключите:**
   - Grove LED Bar к пинам 2, 3
   - Сервопривод к пину 9
   - Arduino к компьютеру через USB

2. **Загрузите:** `src/test/treasure_chest_test.ino`

3. **Откройте Serial Monitor** (115200 бод)

4. **Введите команды:**
   ```
   HELP         ← Получить справку
   AUTO         ← Автоматический режим
   STATUS       ← Показать статус
   SUCCESS      ← Симуляция успеха
   FAIL_EMG     ← Провал по мышцам
   FAIL_EEG     ← Провал по ЭЭГ
   RESET        ← Перезапустить
   ```

### Вариант 2: Работа с реальными датчиками

1. **Подключите все компоненты** по схеме в `docs/hardware/connection-schema.md`

2. **Разместите электроды:**
   - **ЭМГ:** над левой и правой бровью (2 см выше)
   - **ЭЭГ:** Fpz (над переносицей) + за ухом (референт)

3. **Загрузите:** `src/main/treasure_chest.ino`

4. **Откройте Bitronics Lab** или Serial Monitor (112500 бод)

5. **Нажмите RESET** на Arduino

6. **Пройдите калибровку** (30 секунд):
   - Сидите спокойно
   - Лицо расслаблено
   - Глаза открыты

7. **Выполните тренинг** (3 минуты):
   - Расслабьте лоб → крышка открывается
   - Решайте арифметические задачи → бета-ритм растёт

## 📊 Критерии успеха

| Условие | Требование | Индикация |
|---------|-----------|-----------|
| **Крышка** | Открыта ≥54° (60% от 90°) | LED Bar показывает угол |
| **Бета-ритм** | > порога ≥60% времени | Зелёные светодиоды |
| **Общий успех** | Оба условия выполнены | Все светодиоды горят, зелёные мигают 5 раз |
| **Неудача** | Хотя бы одно не выполнено | Красные светодиоды мигают 3 раза |

```cpp

// Коэффициенты порогов (1.2 = легче, 1.5 = сложнее)
```

## 🔋 Автономная работа

**Устройство может работать без компьютера!**

### Что нужно:
- Power Bank 5V 2A (или батарейный отсек 4xAA)
- Подключить к USB порту Arduino или к Vin+GND

### Как работает:
1. Загрузите `src/main/treasure_chest.ino`
2. Подключите все датчики
3. Подайте питание от Power Bank
4. Нажмите RESET
5. Устройство автоматически проведёт калибровку и тренинг

### Индикация результата:
- ✅ **УСПЕХ:** Все 10 светодиодов горят, зелёные мигают 5 раз
- ❌ **НЕУДАЧА:** Красные светодиоды мигают 3 раза

## 🐛 Диагностика проблем

| Проблема | Решение |
|----------|---------|
| Серво дёргается | Отдельное питание 5V 1A+, увеличить SERVO_INTERVAL до 300 |
| Крышка не открывается | Уменьшить EMG_THRESHOLD_K до 1.2, проверить контакт электродов |
| LED Bar не светится | Проверить подключение D1→2, D2→3, установить библиотеку |
| Шум на сигналах | Обезжирить кожу, использовать гель, проверить заземление |
| Всегда неудача | Провести тренировочные сеансы, уменьшить пороги |
