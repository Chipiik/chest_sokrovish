# Инструкция по тестированию "Сундук сокровищ"

## Подключение оборудования

1. **Grove LED Bar v1.0** → Arduino Mega:
   - DATA → пин 2
   - CLK → пин 3
   - VCC → 5V
   - GND → GND

2. **Сервопривод FS5103B** → Arduino Mega:
   - Сигнал (жёлтый/оранжевый) → пин 9 (PWM)
   - Питание (красный) → 5V
   - Земля (коричневый/чёрный) → GND

## Запуск программы

1. Откройте `treasure_chest_test.ino` в Arduino IDE
2. Выберите плату: **Arduino Mega or Mega 2560**
3. Выберите порт: **COM-порт вашей Arduino**
4. Загрузите скетч
5. Откройте **Serial Monitor** на скорости **115200 бод**

## Команды управления

Введите одну из команд в Serial Monitor:

| Команда | Описание |
|---------|----------|
| `SUCCESS` | **Успешный сценарий**: крышка откроется, светодиоды загорятся |
| `FAIL_EMG` | **Провал по ЭМГ**: напряжение мышц лба, крышка закроется |
| `FAIL_EEG` | **Провал по ЭЭГ**: нет бета-ритма, крышка откроется но без "камней" |
| `A CYCLE` | **Авто-цикл**: полная симуляция (30 сек калибровка + 3 мин тренинг) |
| `I` | **Инфо**: показать текущее состояние системы |
| `H` | **Справка**: показать все команды |
| `R` | **Сброс**: перезапустить систему |

## Визуализация графиков

Для просмотра графиков в реальном времени:
1. Откройте **Serial Plotter** (вместо Serial Monitor)
2. Введите команду `A CYCLE` или любой сценарий
3. Наблюдайте 4 графика:
   - **ЭМГ** (синий) - напряжение мышц
   - **Бета-ритм** (зелёный) - мощность 0.0-1.0
   - **Угол крышки** (красный) - 0-90 градусов
   - **Статус расслабления** (фиолетовый) - 1=расслаблен, 0=напряжён

## Ожидаемое поведение

### SUCCESS
- Grove LED Bar: постепенно загорается до 10 светодиодов
- Серво: плавно открывается до ~90°
- Результат: ★ УСПЕХ!

### FAIL_EMG  
- Grove LED Bar: гаснет или мигает
- Серво: остаётся закрытым или закрывается
- Результат: ✗ ПРОВАЛ (напряжение мышц)

### FAIL_EEG
- Grove LED Bar: загорается (крышка открылась)
- Серво: открывается до ~90°
- Результат: ✗ ПРОВАЛ (недостаточный бета-ритм)

### A CYCLE
- Автоматически проходит калибровку (30 сек)
- Затем тренинг (3 мин) с эмуляцией хороших показателей
- В конце показывает результат

## Работа без компьютера

**ДА, устройство может работать автономно от Power Bank!**

Для этого нужно:
1. Использовать основную версию `treasure_chest.ino` (с реальными датчиками)
2. Подключить Power Bank 5V 2A к USB порту Arduino
3. Устройство автоматически проведёт калибровку и тренинг
4. Результат показывается светодиодами

Тестовая версия `treasure_chest_test.ino` требует ввода команд через Serial, поэтому для автономной работы не предназначена.

## Диагностика проблем

| Проблема | Решение |
|----------|---------|
| "Кракозябры" в мониторе порта | Выберите скорость **115200 бод** |
| Серво дёргается | Проверьте питание (нужен мощный USB/Power Bank) |
| LED Bar не горит | Проверьте подключение пинов 2 и 3 |
| Компилляция не удаётся | Установите библиотеку **Grove_LED_Bar** |

## Установка библиотек

В Arduino IDE:
1. **Sketch** → **Include Library** → **Manage Libraries**
2. Найдите **Grove_LED_Bar** от Seeed Studio
3. Нажмите **Install**
4. Библиотека **Servo** уже встроена в Arduino IDE
